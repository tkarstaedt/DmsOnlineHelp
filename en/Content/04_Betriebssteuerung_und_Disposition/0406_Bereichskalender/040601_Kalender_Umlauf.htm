      Calendar: Blocks

      The block calendar displays blocks on a time scale and supports planning, dispatching, and monitoring throughout the operating day.

      Views

      View Purpose
      Day Detailed planning for one operating day (start/end, buffers, overlaps).
      Week Comparison over several days, pattern recognition, holiday shifts.

      Filters & Layers

      Filters narrow down the view to relevant blocks; layers control the visualization.

      • Line, block group, depot, status.
      • Layers: Planned vs. actual, operational reserve, special block.
      • Show/hide vehicle type constraints and user-scheduled vehicles.

      Interactions

      The most important operations in the calendar:

      • Open: Double-click on a block > Block dossier.
      • Context menu: Right-click on block > Edit vehicle types, Create special block, Go to message.
      • Drag & Drop: Move block chronologically (if status is “Planned”).
      • Multi-select: Ctrl+click for collective actions (e.g., set status).

      Conflict & Plausibility Checks

      The calendar warns about contradictory changes.

      • Overlapping block times or invalid sequences.
      • Incompatible vehicle types or lack of availability.
      • Rule violations (e.g., end time < start time).
