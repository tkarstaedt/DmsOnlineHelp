      Dialog "Set Target Location"

      Overview

      In the "Set Target Location" dialog, the authorized user assigns a stabling position in the depot to the selected vehicle (manual dispatching).

      The dialog is called up in the "Vehicle Dispatching" dialog via the Manual Dispatching button.

      Dialog "Set Target Location"

      Dialog Fields

      The dialog contains the following dialog fields:

      Field

      Description

      Vehicle

      Number of the selected vehicle.

      Vehicle Type

      Vehicle type corresponding to the vehicle.

      Location — Information on the planned stabling position.

      Depot

      Depot.

      Place

      Track or storage area.

      Length

      Length of the specified track (informal information).

      Position

      Position of the vehicle on the track: distance between the vehicle and the beginning of the track (calculated value in meters).

      The calculation considers vehicles already stabled on the track and a minimum distance between vehicles.

      Section

      Name of the section on the track, provided the track is divided into sections.

      Dialog "Set Target Location" — Dialog Fields

      Operating Actions

      The following operating actions can be performed in the "Set Target Location" dialog:

      • Set Planned Location
      • Delete Planned Location

      Set Planned Location

      The "Set Planned Location" operating action means manually dispatching a vehicle.

      The assignment can be done:

      • via <Ctrl> + Drag-and-drop
      • in the "Vehicle Dispatching" dialog.

      Result: The assigned stabling position is the vehicle's planned location. The stabling position is displayed in the "Vehicles" view, in the "Dispatch Location" column.

      Manually Dispatch Vehicle via <Ctrl> + Drag-and-drop

      Manually dispatch vehicle via <Ctrl> + Drag-and-drop — Step by step:

      Prerequisite:

      • The graphical view "Depot View" is open.
      • The tabular view "Vehicles" is open.
      1. "Vehicles" view: Left-click on the vehicle to be positioned.
        → The row is highlighted.
      2. Hold down the Ctrl key and the left mouse button and drag the vehicle into the "Depot View" view, onto the desired track.

        Display of the permitted insertion area:

        Display of the non-permitted insertion area:

        Set Planned Location — Drag the vehicle to the planned stabling position using Ctrl+Drag-and-drop

      3. Release the Ctrl key and the mouse button.
        → The vehicle icon is displayed on the track.

        Set Planned Location — Vehicle is set at the planned location. The layout of the vehicle icon marks the stabling position reservation.

      The planned location in the "Depot View" can be changed by moving the vehicle icon (drag-and-drop).

      Manually Dispatch Vehicle in the "Vehicle Dispatching" Dialog

      Manually dispatch vehicle in the "Vehicle Dispatching" dialog — Step by step:

      Prerequisite:

      • The "Vehicle Dispatching" dialog is open.

      1. Vehicle: Select the vehicle to be dispatched.
      2. Left-click on Manual Dispatching.
        → The "Set Target Location" dialog is open.
      3. "Set Target Location" dialog: Depot: Select the depot.
      4. "Set Target Location" dialog: Place: Select the stabling position.
        → Length: The length of the selected stabling position is displayed.

        The position is calculated after confirming the entries with "OK". The position is displayed when the "Set Target Location" dialog is reopened.

      5. "Set Target Location" dialog: Left-click on OK.
        → The "Set Target Location" dialog is closed. The planned location of the vehicle is set.

        In the "Vehicle Dispatching" dialog, the "Depot" and "Stabling Position" fields are filled according to the user entries.

      6. Left-click on Close.
        → The "Vehicle Dispatching" dialog is closed.

      Delete Planned Location

      The "Delete Planned Location" operating action means canceling the stabling position reservation and thus the planned assignment of the vehicle to a stabling position in the depot.

      Delete Planned Location in the "Depot View"

      Delete planned location in the "Depot View" — Step by step:

      1. Context menu of the vehicle > Delete Planned Location.

        Delete Planned Location — Context menu of the vehicle with a planned location.

      2. Pop-up window: Left-click on the Yes button.

        Delete Planned Location — Pop-up window with security question.


        → The stabling position reservation is deleted.

      Delete Planned Location in the "Vehicle Dispatching" Dialog

      Delete planned location in the "Vehicle Dispatching" dialog — Step by step:

      Prerequisite:

      • The "Vehicle Dispatching" dialog is open.
      • The planned location is displayed in the "Depot" and "Stabling Position" fields.
      1. Left-click on the Delete Target Location button.
        → The planned location is deleted. The "Depot" and "Stabling Position" fields are empty.
