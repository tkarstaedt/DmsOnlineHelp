      View “Pull-out list” - Depot departures

      The Pull-out list shows planned, ongoing, and completed pull-outs per operating day and depot; use the context menu to open the block dossier for details.

      Overview

      The Pull-out list is a tabular dispatching view in the DMS client. It focuses on depot exits (pull-outs) and serves as a control instrument for smooth depot traffic.

      Access

      • Menu: View → Data views → Pull-out list
      • Toolbar: Pull-out list button

      Period

      In the function bar, set the operating day and No. of days to include following days. Optionally the view advances automatically when the operating day changes.

      Content and columns

      Standard columns (generic) in the Pull-out list:

      Column Description
      Vehicle Current vehicle for the block segment.
      Block Block number.
      Depot Code of the starting depot of the block segment.
      Start (planned) Operating day and time when the block segment is planned to start (pull-out/start of service).
      State Current state of the block segment; matches the state of the related block.
      Start (actual) Actual start (pull-out/start of service); set by positioning at the depot exit.

      Additional columns (project-specific)

      • Operating day, Block type
      • Pull-out time (planned/actual) and Deviation [min]
      • Duty, Pull-out duty, To (destination of first duty)
      • Start vehicle, Planned vehicle, User-scheduled vehicle
      • Pull-out parking (planned), Pull-out segment (planned), Current parking, Current segment
      • Note, Turn-back, Swap duty

      Behavior and logic

      • The Pull-out list shows all block segments that start at a depot.
      • Only the first starting segment of a block has no Turn-back flag; further starting segments of the same block get Turn-back.
      • If a vehicle performs multiple segments, the assignment is propagated; state In progress is copied from the block to related segments.

      Typical actions

      • Edit block: context menu on an entry with state Planned → Edit → Block dossier dialog.

      Report

      The Pull-out list report summarizes all depot exits with planned and actual times, grouped by operating day.
