      Filter records

      Use filters in tabular views of PEAK.DMS to restrict the displayed records to a subset. This hides irrelevant items so you can focus on what matters.

      Reference: Fundamentals of sorting, searching, and column adjustments are described in Views — data presentation and Search records.

      Filter types

      • Basic filters: work per column via a filter row, comparable to filters in MS Excel.
      • Complex filters: combine multiple conditions and links in a dedicated area above the table.

      Use basic filters

      1. In the toolbar, click the icon for Show filter row. A blank row appears below the column headers.
      2. In the desired column, enter the filter criterion in the filter row. The table is filtered immediately.

      Figure: Activate basic filters and enter a criterion

      Select operators per column

      Available comparison operators depend on the column’s data type (text, number, date). Open the operator menu via the icon at the left edge of the filter cell.

      • Text columns: contains, begins with, ends with, equals, not equal
      • Numeric columns: =, ≠, <, ≤, >, ≥, range
      • Date/Time columns: on, before, after, between

      Figure: Operators for text values

      Figure: Operators for numeric values

      Manage filters in the column header

      When a column is filtered, a filter icon appears in the header. Click it to edit details.

      • Change the operator in the dropdown.
      • Adjust the criterion in the input field.
      • Remove filter to clear the column.

      Use complex filters

      1. Click Show complex filter. The filter area appears above the table.
      2. Existing basic or complex filters are displayed there graphically and can be extended or modified.

      Figure: Complex filter area

      Use the visual filter editor

      1. Click Edit filter to open the visual editor.
      2. Define the filter by clicking groups, conditions, columns, operators, and values.
      3. Click OK to apply. The definition appears in the filter area and is applied to the table.

      Figure: Visual filter editor

      Structure of complex filters

      • Groups: combine conditions with AND or OR. Groups form the nodes of the tree.
      • Conditions: entries under a group. Each condition has a column, operator, and value.
      • Click the + behind a group to add a condition.
      • Click the x on a group or condition to remove it.
      • Click Column, Operator, or Value to change the selection or input.
