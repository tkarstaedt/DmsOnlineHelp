      Correcting Vehicle Location

      With the Set Location , you can manually correct the location of a vehicle in the , move a vehicle to a parking position, or remove it from the .

      Opening the Dialog

      • Toolbar Dialog → Set Location
      • Or: of a vehicle → Set Location…

      Correcting Vehicle Location

      1. Select the Vehicle in the selection list. The is displayed next to the field.
      2. Select the Depot area where the vehicle should be placed.
      3. Select the Parking Position in the Location field. Below, the track length is displayed (if multiple vehicles can be parked there).
      4. In the Position field, enter the distance in meters from the beginning of the track. The system automatically suggests an optimal position behind already stabled vehicles.
      5. Click Apply to save the change.
      6. Click Close to exit the .

      Removing Vehicle from Depot

      1. Select the vehicle.
      2. Set the Depot field to <Netz> (Network).
      3. Confirm with Apply → the vehicle is removed from the .
      4. Exit the with Close.

      Special Features

      • If the was opened via the of a vehicle, the vehicle is preselected.
      • When a vehicle is selected, current data (Location, Position) is automatically loaded.
      • If the vehicle is part of a Traction, this is displayed. The Decouple button can be used to resolve the before correcting the location.
      • The track length is only displayed if multiple vehicles are allowed there and it is not a storage space.
      • For parking positions for only one vehicle or storage space, no position entry is possible.
      • If a position is selected where there is no space, the system displays a warning or error message with the blocking vehicles.
