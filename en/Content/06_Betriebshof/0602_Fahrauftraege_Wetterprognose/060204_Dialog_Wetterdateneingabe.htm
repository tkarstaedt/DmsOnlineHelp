      Dialog "Weather Data Entry"

      In the "Weather Data Entry" dialog, you enter or change weather values for the depot in PEAK.DMS to supplement or override forecasts. Typical entries include temperature, precipitation, wind, and special situations like frost or storm.

      Opening the dialog

      1. From the "Weather Forecast" view: Select New… or Edit…. For background information, see View "Weather Forecast".
      2. Alternatively: Via administration or service menus (project-specific).

      Fields (Default)

      The available fields may vary depending on the project. In the default configuration, the following inputs are available:

      Field Description
      Date Calendar day of the measurement/forecast.
      Time Time point or start of time window.
      Temperature [°C] Air temperature at the location.
      Precipitation [mm/h] Intensity per hour.
      Precipitation type e.g., rain, snow, sleet.
      Wind [km/h] Average speed.
      Gusts [km/h] Maximum gust speed.
      Humidity [%] Relative humidity.
      Air pressure [hPa] Station or reduced pressure (project-specific).
      Special situation Notification flag: Frost, storm, slippery conditions, etc.
      Note Short comment for dispatching/service.
      Source Manual, External (API/Import), or Mixed.
      Valid for location Depot/Station (select if multiple locations).

      Options and Validity

      Control how entered values affect the system.

      • Activate override: Manual values overwrite forecasts in the selected time period.
      • Define time period:Specify a single point in time or an interval (from/to).
      • Confidence level (optional):Mark data quality (high/medium/low).

      Validation and Notes

      The entries are checked for plausibility.

      • Units: Use the specified units (e.g., °C, km/h, mm/h).
      • Value ranges: Temperature -40…+50 °C, Wind/Gusts 0…200 km/h (project-specifically adjustable).
      • Consistency: Gusts ≥ Wind speed; Interval end > start.

      Typical work steps

      1. Open the dialog (see Dialog "Weather Data Entry").
      2. Select Date, Time (or time period) and the Location.
      3. Enter Temperature, Precipitation, Wind/Gusts; set Special situation if necessary.
      4. Activate Override if needed and save with OK.

      Effects in operation

      Entered values are incorporated into views and planning.

      • Weather forecast: Display/profile is adjusted for the time period (see View "Weather Forecast").
      • Dispatching: Notes for pull-outs, shunting trips, and cleaning are supported (e.g., in case of frost/storm).
      • E-bus preconditioning: Plans for charging and climate phases consider temperature/wind.

      Saving and closing

      • OK: Save data and close dialog.
      • Cancel: Discard changes.
