      “Repositioning Move” Dialog (Create Repositioning Move)

      In the “” , you create or modify a planned vehicle movement within the (from A to B) in PEAK.DMS, including time, priority, and reason.

      Opening the Dialog

      1. From the : Right-click or vehicle → Create …. For introduction, see “Depot View”.
      2. From the “Parking Positions” View: Select → > Create …. For working with the list, see “Parking Positions” View.
      3. From (if enabled): New order of type “”. Find further information in “Driving Orders” View.

      Prerequisites

      • Write permissions for .
      • Source and target parking positions exist and are not permanently blocked.
      • Vehicle is available (not bound to /; project-specific).

      Fields and Options (Standard)

      The actual fields may vary project-specifically. Typical entries in the “” :

      Field Description
      Vehicle Selection of the vehicle to be moved (number/type).
      Source (from) Current or area.
      Target (to) Desired or area.
      Start Planned start (date/time).
      End (optional) Planned end/deadline.
      Priority Urgency level for /processing.
      Reason Short reason (e.g., duty lineup plan, , reserve).
      Note (optional) Internal comment for /Service.
      Execution Type Manual (driver) or internal (operational process; project-specific).

      Creating a Repositioning Move

      1. Open the “” (see “Repositioning Move” Dialog (Create Repositioning Move)) and select Vehicle.
      2. Check Source (from) and select Target (to).
      3. Set Start (and optional End) and Priority.
      4. Enter Reason and Note, if applicable.
      5. Select the Execution Type and confirm with OK.

      Editing or Canceling a Repositioning Move

      Existing can be modified or canceled.

      • Edit: Select in the list/display → Edit… → adjust fields → OK.
      • Cancel: Select → Cancel/Reject → specify reason → OK.

      Notes and Best Practices

      • Observe blocks and (, , , reserve).
      • Check for conflicts with stabling or duty planning (project-specific ).
      • E-Bus: Select a target that is charge-capable if possible; observe charging window and SOC.

      Save and Close

      • OK: Save and close .
      • Cancel: Discard changes.
