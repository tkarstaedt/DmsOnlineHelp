      Display electrical process data

      The electrical process data shows dynamic information on the current state of charge of an electric vehicle. It is displayed on the "Electrical process data" tab of the vehicle dossier and cannot be edited. The data originates from the onboard electronics or telematics and is used for monitoring and planning.

      Displayed fields

      In the default configuration, the following fields are displayed:

      Field Description
      Current SOC [%] Current state of charge of the battery in percent.
      Current SOC [kWh] Currently usable energy of the traction battery in kWh.
      Remaining range without block reference Calculated range (km) when no planned block is considered.
      Remaining range with block reference Remaining range (km) with respect to the currently dispatched block.
      Charging phase (checkbox) Checked when a charging phase is active (charging, preconditioning, balancing).
      Charging start time Point in time when charging (if scheduled) should start.
      Charge completion time (estimated) Estimated time when the battery reaches the target SOC (full charge).

      Usage

      • Planning: Use the process data as a starting point to create charging orders (see Create Charging Order).
      • Monitoring: Compare the current SOC with the thresholds stored in the system. In case of deviations, corresponding alarms are triggered (see Alarming & Thresholds).
      • History: Some systems log the SOC progression over time to analyze battery degradation (project-specific).

      Notes

      • The process data is read-only; adjustments are made via charging orders and stabling allocation.
      • Inconsistencies (e.g., missing data) may indicate a fault in the telematics or the vehicle.
