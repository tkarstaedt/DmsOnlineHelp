      Switch user, sign out, and change password

      Switch user

      With the action Switch user, you sign in a new user while the currently signed‑in user is signed out at the same time. The switch is performed via the Switch user dialog.

      Menu bar: Choose File > Switch user.

      Figure 4: “Switch user” dialog

      1. In New user, enter the username to sign in.
      2. In Password, enter the password of the new user.
      3. Click OK.

      The currently signed‑in user is signed out. Their last used window layout is saved automatically and can be loaded again at the next sign‑in. The application window remains open with an empty display area. Any open instances of the application window are closed.

      After successful validation of username and password, the new user is signed in.

      Optional: Enable the Use current profile checkbox if you want to keep the last used window layout.

      Sign out

      With the action Sign out, you sign the currently signed‑in user out of the system.

      1. In the menu bar, choose File > Sign out. The PEAK.DMS Security window opens.
      2. Click OK.

      The PEAK.DMS Security window closes. The user is signed out and their last used window layout is saved automatically. The application window remains open with an empty display area, and the User sign‑in dialog is displayed.

      Change password

      With the action Change password, you replace your current password with a new one. The change is performed in the Change password dialog.

      Menu bar: Choose File > Change password.

      Figure 5: “Change password” dialog

      1. In Old password, enter your current password.
      2. In New password, enter the new password.
      3. Repeat the entry in Repeat password.
      4. Click OK.

      The dialog closes. The new password will be active at your next sign‑in.
