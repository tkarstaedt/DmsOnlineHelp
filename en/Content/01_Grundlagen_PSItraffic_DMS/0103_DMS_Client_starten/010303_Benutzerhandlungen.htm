Benutzer wechseln, abmelden und Kennwort ändern

Benutzer ummelden

Mit der Bedienhandlung Benutzer ummelden melden Sie einen neuen Benutzer an, während der aktuell angemeldete Benutzer gleichzeitig abgemeldet wird. Die Ummeldung erfolgt über den Dialog Benutzerummeldung .

Menüleiste: Wählen Sie Datei > Ummelden .

Abbildung 4: Dialog „Benutzerummeldung“

  1. Geben Sie im Feld Neuer Benutzer den Benutzernamen des anzumeldenden Benutzers ein.
  2. Geben Sie im Feld Kennwort das Kennwort des neuen Benutzers ein.
  3. Klicken Sie auf OK .

Der aktuell angemeldete Benutzer wird abgemeldet. Sein zuletzt verwendetes Fensterlayout wird automatisch gespeichert und kann bei der nächsten Anmeldung wieder geladen werden. Das Anwendungsfenster bleibt mit leerem Anzeigebereich geöffnet. Eventuell geöffnete Instanzen des Anwendungsfensters werden geschlossen.

Nach erfolgreicher Prüfung von Benutzername und Kennwort wird der neue Benutzer angemeldet.

Optional: Aktivieren Sie die Checkbox Aktuelles Profil verwenden , wenn Sie das zuletzt verwendete Fensterlayout übernehmen möchten.

Benutzer abmelden

Mit der Bedienhandlung Benutzer abmelden melden Sie den aktuell angemeldeten Benutzer vom System ab.

  1. Wählen Sie in der Menüleiste Datei > Abmelden . Das Fenster PEAK.DMS-Sicherheit wird geöffnet.
  2. Klicken Sie auf OK .

Das Fenster PEAK.DMS-Sicherheit wird geschlossen. Der Benutzer wird abgemeldet und sein zuletzt verwendetes Fensterlayout automatisch gespeichert. Das Anwendungsfenster bleibt mit leerem Anzeigebereich geöffnet, und der Dialog Benutzeranmeldung wird angezeigt.

Kennwort ändern

Mit der Bedienhandlung Kennwort ändern ersetzen Sie Ihr aktuelles Kennwort durch ein neues. Die Änderung erfolgt im Dialog Kennwort ändern .

Menüleiste: Wählen Sie Datei > Kennwort ändern .

Abbildung 5: Dialog „Kennwort ändern“

  1. Geben Sie im Feld Altes Kennwort Ihr aktuelles Kennwort ein.
  2. Geben Sie im Feld Neues Kennwort das neue Kennwort ein.
  3. Wiederholen Sie die Eingabe im Feld Kennwortwiederholung .
  4. Klicken Sie auf OK .

Der Dialog wird geschlossen. Das neue Kennwort ist bei Ihrer nächsten Anmeldung aktiv.
