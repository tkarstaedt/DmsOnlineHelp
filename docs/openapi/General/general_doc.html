Developer Documentation: General OpenAPI Interfaces

General usage instructions, security concepts, and retrieval methods for the PEAK.X platform.

1. Introduction

1.1 Product Branding

The software platform formerly known as "PSItraffic" is hereinafter branded as PEAK.X. It serves as the foundation for:

  • PEAK.DMS: Depot Management System
  • PEAK.TMS: Train Management System
  • PEAK.ITCS: Intermodal Transport Control System

The term "PEAK.X" refers to the system capabilities generally, unless a specific product is named.

1.2 About This Manual

This manual provides a comprehensive overview of the OpenAPI interfaces used within the PEAK.X platform. It is a central resource for developers integrating these interfaces and administrators configuring them.

2. OpenAPI Interfaces

The OpenAPI interfaces are organized into modular components. Below is an overview of key modules available in PEAK.X.

3. Security & Auth

Authentication

  • None: Only for initial testing (not recommended).
  • Basic Auth: Base64-encoded username:password in Header. Requires a PEAK.X user with API rights.

Encryption

Communication can be HTTP or HTTPS.

Important: HTTPS is highly recommended for production to protect credentials and tokens.

4. Data Retrieval Methods

PEAK.X supports multiple methods to balance latency and complexity.

4.1 Polling

The simplest method: Client sends GET request, Server responds immediately. Good for occasional updates.

Client --(GET)--> PEAK.X
Client <--(200 OK)-- PEAK.X

4.2 Long Polling (Stream)

Connection stays open (default 20s) until data changes. Reduces request frequency.

  • Uses clientId to track state.
  • If clientId is unknown: Server returns full dataset.
  • If clientId is known: Server returns only changes (deltas).
GET .../stream?clientId={clientId}

4.3 Callback (Push)

Asynchronous real-time updates. Client subscribes, Server pushes updates via POST to a callback URL.

Subscribe POST /resource/subscription
Returns subscriptionId.
Update Server sends POST to your URL when data changes.
