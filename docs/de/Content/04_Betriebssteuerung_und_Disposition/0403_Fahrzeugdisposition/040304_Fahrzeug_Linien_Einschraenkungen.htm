Dialog "Fahrzeug-Linien-Einschränkungen"

Mit dem Dialog "Fahrzeug-Linien-Einschränkungen" können Sie für ein Fahrzeug eine oder mehrere Linien festlegen, auf denen es eingesetzt werden muss. Die Einschränkungen wirken auf die automatische und manuelle Disposition von Umläufen mit Linienbezug.

Übersicht

Der Dialog "Fahrzeug-Linien-Einschränkungen" ermöglicht die manuelle Festlegung von Linienrestriktionen für einzelne Fahrzeuge. Diese definieren, auf welchen Linien ein Fahrzeug disponiert werden darf.

Jede Einschränkung ist zeitlich befristet und kann mit einem optionalen Grund versehen werden. Das System zeigt zudem alle bestehenden Werkstatt- und Versorgungsaufträge zum ausgewählten Fahrzeug an.

Dialog aufrufen

  1. Öffnen Sie das Menü Bearbeiten → Fahrzeug-Linien-Einschränkungen.

Dialog "Fahrzeug-Linien-Einschränkungen"

Dialogfelder

Der Dialog enthält die folgenden Eingabefelder:

Feld Beschreibung
Fahrzeug Fahrzeugnummer des Fahrzeugs, für das Linieneinschränkungen erfasst werden.
Das System zeigt alle Werkstatt- und Versorgungsaufträge zu diesem Fahrzeug an.
Linie Auswahlliste mit allen verfügbaren Linien.
Wählen Sie die Linie aus, auf der das Fahrzeug eingesetzt werden darf.
Gültig bis (Bis) Zeitpunkt, bis wann die Linieneinschränkung gilt.
Standardwert: Ende des nächsten Betriebstages (kann manuell überschrieben werden).
Grund Optionales Freitextfeld zur Erfassung des Grundes bzw. Anlasses für die Einschränkung.

Dialog "Fahrzeug-Linien-Einschränkungen" — Dialogfelder

Bedienhandlungen

Linieneinschränkung erfassen

Linieneinschränkung erfassen — Schritt für Schritt:

  1. Dialog Fahrzeug-Linien-Einschränkungen über Menü Bearbeiten öffnen.
  2. Feld Fahrzeug: Fahrzeugnummer eingeben oder auswählen.
    → Das System zeigt alle Werkstatt- und Versorgungsaufträge zum Fahrzeug an.
  3. Feld Linie: Gewünschte Linie aus der Auswahlliste wählen.
  4. Feld Gültig bis: Vorgeschlagenes Enddatum prüfen und bei Bedarf anpassen.
  5. Feld Grund (optional): Grund für die Einschränkung erfassen.
  6. Schaltfläche Übernehmen klicken.
    → Die Linieneinschränkung ist gespeichert.
    → Das System stellt automatisch eine weitere leere Zeile bereit.
  7. Bei Bedarf weitere Linieneinschränkungen erfassen (Schritte 3-6 wiederholen).
  8. Dialog schließen.

Mehrere Einschränkungen erfassen

Sie können mehrere Linieneinschränkungen für ein Fahrzeug parallel erfassen. Nach jeder Bestätigung mit Übernehmen wird automatisch eine neue Eingabezeile zur Verfügung gestellt.

Nach erfolgter Eingabe können Sie für ein anderes Fahrzeug Linieneinschränkungen erfassen.

Wirkung in der Disposition

Die Linieneinschränkungen wirken sich wie folgt auf die Fahrzeug- und Umlaufdisposition aus:

Grundregeln

  • Die Linieneinschränkungen gelten für alle Umlaufstücke eines Umlaufs.
  • Enthält ein Umlauf mehrere verschiedene Linien und ist das Fahrzeug nur auf eine dieser Linien eingeschränkt, wird das Fahrzeug nicht auf diesen Umlauf disponiert.

Wichtig: Die Einschränkung wirkt umlaufübergreifend. Wenn ein Fahrzeug auf Linie XX eingeschränkt wurde und ein Umlauf Umlaufstücke für die Linie XX und YY enthält, dann wird das Fahrzeug für diesen Umlauf nicht eingesetzt.

Berücksichtigung von Werkstatt- und Versorgungsaufträgen

  • Werkstatt- und Versorgungsaufträge werden stets uneingeschränkt beachtet.
  • Ein Fahrzeug mit Linieneinschränkung wird nur dann auf einen entsprechenden Umlauf disponiert, wenn vor Umlaufende kein Werkstatt- oder Versorgungsauftrag beginnt.

Zusammenwirken mit weiteren Dispositionsregeln

Linienrestriktionen wirken zusammen mit weiteren Regeln aus der Disposition und Kursaufstellplanung:

  • Zulässige Fahrzeugtypen
  • Wunschfahrzeuge
  • Regeln aus der Kursaufstellplanung

Beispiel

Szenario:

Ein Fahrzeug ist ausschließlich für den Einsatz auf Linie 20 zugelassen.

Ergebnis:

  • Umläufe, die nur Linie 20 enthalten → Fahrzeug wird disponiert
  • Umläufe, die sowohl Linie 20 als auch Linie 21 enthalten → Fahrzeug wird ausgeschlossen

Hinweise

Nutzen Sie das Feld "Grund", um die Nachvollziehbarkeit von Linieneinschränkungen für andere Disponenten zu erhöhen (z. B. "Fahrereinweisung Linie 20", "Testfahrten").

  • Mehrere Linieneinschränkungen können für ein Fahrzeug gleichzeitig aktiv sein.
  • Jede Einschränkung verfügt über eine eigene Gültigkeitsdauer.
  • Abgelaufene Einschränkungen werden automatisch nicht mehr berücksichtigt.
