Dialog „Zielstandort setzen“

Übersicht

Im Dialog „Zielstandort setzen“ weist der berechtigte Benutzer dem ausgewählten Fahrzeug einen Stellplatz auf dem Betriebshof zu (manuelle Disposition).

Der Dialog wird im Dialog „Fahrzeugdisposition" über die Schaltfläche Manuell Disponieren aufgerufen.

Dialog "Zielstandort setzen"

Dialogfelder

Der Dialog beinhaltet die folgenden Dialogfelder:

Feld

Beschreibung

Fahrzeug

Nummer des ausgewählten Fahrzeugs.

Fahrzeugtyp

Fahrzeugtyp zum Fahrzeug.

Standort — Informationen zum geplanten Stellplatz.

Betriebshof

Betriebshof.

Ort

Spur oder Stauraum.

Länge

Länge der angegebenen Spur (informelle Angabe).

Position

Position des Fahrzeugs auf der Spur: Abstand zwischen dem Fahrzeug und dem Anfang der Spur (berechnete Angabe in Metern).

Bei der Berechnung werden die auf der Spur bereits abgestellten Fahrzeuge und ein Mindestabstand zwischen den Fahrzeugen berücksichtigt.

Abschnitt

Name des Abschnitts auf der Spur, sofern die Spur in Abschnitte unterteilt ist.

Dialog "Zielstandort setzen" — Dialogfelder

Bedienhandlungen

Im Dialog "Zielstandort setzen" können die folgenden Bedienhandlungen durchgeführt werden:

  • Geplanten Standort setzen
  • Geplanten Standort löschen

Geplanten Standort setzen

Die Bedienhandlung "Geplanten Standort setzen" bedeutet, ein Fahrzeug manuell zu disponieren.

Die Zuweisung kann erfolgen:

  • via <Strg> + Drag-and-Drop
  • im Dialog "Fahrzeugdisposition".

Ergebnis: Der zugewiesene Stellplatz ist geplanter Standort des Fahrzeugs. Der Stellplatz wird in der Sicht "Fahrzeuge", Spalte "Dispo-Stellplatz" angezeigt.

Fahrzeug manuell disponieren via <Strg> + Drag-and-Drop

Fahrzeug manuell disponieren via <Strg> + Drag-and-Drop — Schritt für Schritt:

Voraussetzung:

  • Die grafische Sicht "Betriebsbild" ist geöffnet.
  • Die tabellarische Sicht "Fahrzeuge" ist geöffnet.
  1. Sicht "Fahrzeuge": Linksklick auf das zu positionierende Fahrzeug.
    → Die Zeile ist markiert.
  2. Strg-Taste und linke Maustaste gedrückt halten und das Fahrzeug in die Sicht "Betriebsbild", auf die gewünschte Spur ziehen.

    Darstellung des zulässigen Einfügebereichs:

    Darstellung des nicht zulässigen Einfügebereichs:

    Geplanten Standort setzen — Fahrzeug auf den geplanten Stellplatz per Strg+Drag-and-Drop ziehen

  3. Strg-Taste und Maustaste loslassen.
    → Das Fahrzeugsymbol wird auf der Spur angezeigt.

    Geplanten Standort setzen — Fahrzeug ist auf dem geplanten Stellplatz gesetzt. Das Layout des Fahrzeugsymbols kennzeichnet die Stellplatzreservierung.

Der geplante Standort in der Sicht "Betriebsbild" kann durch Verschieben des Fahrzeugsymbols geändert werden (Drag-and-Drop).

Fahrzeug manuell disponieren im Dialog "Fahrzeugdisposition"

Fahrzeug manuell disponieren im Dialog "Fahrzeugdisposition" — Schritt für Schritt:

Voraussetzung:

  • Der Dialog "Fahrzeugdisposition" ist geöffnet.

  1. Fahrzeug: Das zu disponierende Fahrzeug auswählen.
  2. Linksklick auf Manuell disponieren.
    → Der Dialog "Zielstandort setzen" ist geöffnet.
  3. Dialog "Zielstandort setzen": Betriebshof: Betriebshof auswählen.
  4. Dialog "Zielstandort setzen": Ort: Stellplatz auswählen.
    → Länge: Die Länge des ausgewählten Stellplatzes wird angezeigt.

    Die Position wird nach Bestätigung der Eingaben mit "OK" berechnet. Die Position wird angezeigt bei erneutem Öffnen des Dialogs "Zielstandort setzen".

  5. Dialog "Zielstandort setzen": Linksklick auf OK.
    → Der Dialog "Zielstandort setzen" ist geschlossen. Der geplante Standort des Fahrzeugs ist gesetzt.

    Im Dialog "Fahrzeugdisposition" sind die Felder "Betriebshof" und "Stellplatz" der Benutzereingaben entsprechend ausgefüllt.

  6. Linksklick auf Schließen.
    → Der Dialog "Fahrzeugdisposition" ist geschlossen.

Geplanten Standort löschen

Die Bedienhandlung "Geplanten Standort löschen" bedeutet, die Stellplatzreservierung und somit die geplante Zuordnung des Fahrzeugs zu einem Stellplatz auf dem Betriebshof aufzuheben.

Geplanten Standort im "Betriebsbild" löschen

Geplanten Standort im "Betriebsbild" löschen — Schritt für Schritt:

  1. Kontextmenü des Fahrzeugs > Geplanten Standort löschen.

    Geplanten Standort löschen — Kontextmenü des Fahrzeugs mit geplantem Standort.

  2. Pop-up-Fenster: Linksklick auf die Schaltfläche Ja.

    Geplanten Standort löschen — Pop-up-Fenster mit Sicherheitsfrage.


    → Die Stellplatzreservierung ist gelöscht.

Geplanten Standort im Dialog "Fahrzeugdisposition" löschen

Geplanten Standort im Dialog "Fahrzeugdisposition" löschen — Schritt für Schritt:

Voraussetzung:

  • Der Dialog "Fahrzeugdisposition" ist geöffnet.
  • Der geplante Standort wird in den Feldern "Betriebshof" und "Stellplatz" angezeigt.
  1. Linksklick auf die Schaltfläche Zielstandort löschen.
    → Der geplante Standort ist gelöscht. Die Felder "Betriebshof" und "Stellplatz" sind leer.
