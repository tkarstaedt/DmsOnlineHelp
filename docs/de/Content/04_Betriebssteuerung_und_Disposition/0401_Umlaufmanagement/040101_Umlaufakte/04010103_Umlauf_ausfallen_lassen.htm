Umlauf ausfallen lassen

Die Funktion Umlauf ausfallen lassen dient dazu, einen geplanten Umlauf aktiv aus dem Betriebsablauf und der automatischen Disposition zu entfernen.

Umlauf ausfallen lassen

Voraussetzung

Der Menüeintrag bzw. die Schaltfläche Ausfallen lassen erscheint nur, wenn der Umlauf sich im Zustand Geplant befindet und noch nicht begonnen hat.

Vorgehen

  1. Öffnen Sie den Dialog Dialog „Umlaufakte“ (z. B. über die Sicht „Umlaufliste“).
  2. Betätigen Sie die Schaltfläche Ausfallen lassen.
  3. Bestätigen Sie die Sicherheitsabfrage, ob der Umlauf wirklich ausfallen soll, mit Ja.

Konsequenzen und Auswirkungen

Wird ein Umlauf auf Ausgefallen gesetzt, hat dies folgende weitreichende Konsequenzen:

  • Der Umlauf erhält den Zustand Ausgefallen.
  • Der Umlauf wird bei der Disposition nicht mehr berücksichtigt.
  • Fahrzeugzuordnungen werden entfernt:
    • Ein Wunschfahrzeug wird entfernt.
    • Ein aktuell zugewiesenes Fahrzeug wird entfernt.
    • Ein geplantes Fahrzeug (Soll-Fahrzeug) wird entfernt.
  • Auf der Sicht „Fahraufträge“ anzeigen wird der ausgefallene Umlauf nicht mehr angezeigt.
  • War der Umlauf per Vorbelegung auf eine Spur gezogen, wird er von der Spur entfernt.
  • Der ausgefallene Umlauf wird nicht mehr im operativen Kursaufstellplan oder in Vorbelegungsdialogen angezeigt.
  • Es wird eine entsprechende Meldung erzeugt.
  • Melden sich Fahrzeuge über das RBL/ITCS auf ausgefallene Umläufe an, wird ebenfalls eine Meldung erzeugt.

Umlauf reaktivieren

Ein ausgefallener Umlauf kann manuell wieder in den Status Geplant zurückgesetzt werden, sofern die Bedingungen erfüllt sind [9].

Voraussetzung

Eine Reaktivierung eines Umlaufs ist nur möglich, wenn folgende Bedingungen erfüllt sind:

  • Der Umlaufstatus ist Ausgefallen.
  • Die geplante Startzeit des Umlaufs liegt in der Zukunft.

Vorgehen

  1. Öffnen Sie den Dialog „Umlaufakte“.
  2. Betätigen Sie die Schaltfläche Reaktivieren.
  3. Bestätigen Sie die Nachfrage, ob der Umlauf wirklich stattfinden soll.

Konsequenzen

  • Der Umlaufstatus wechselt zurück zu Geplant.
  • Die Schaltfläche heißt wieder Ausfallen lassen.
  • Es wird eine entsprechende Meldung erzeugt.
  • Der Umlauf steht der Disposition wieder zur Verfügung.

Verwandte Themen

  • Dialog „Umlaufakte“
  • Manuelle Umlaufdisposition
  • Grafische Sicht „Aufstellplan“
