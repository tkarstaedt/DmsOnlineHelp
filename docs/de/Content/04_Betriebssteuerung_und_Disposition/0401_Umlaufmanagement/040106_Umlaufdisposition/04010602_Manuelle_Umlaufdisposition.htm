Manuelle Umlaufdisposition

Starten Sie die Umlaufdisposition für einen Betriebshof, Betriebstag und eine Ausrückphase. Das System ordnet Umläufe anhand der aktuellen Hofsituation zu; das Ergebnis sehen Sie im Aufstellplan und im Betriebsbild.

Dialog öffnen

  1. Symbolleiste Dialog → Umlaufdisposition
  2. oder Menü Bearbeiten → Umlaufdisposition

Filterkriterien

  • Betriebshof (nur Höfe mit Dispositionsmodus Freie Aufstellung)
  • Betriebstag
  • Ausrückphase

Disposition ausführen

  1. Klicken Sie auf Disposition starten.
  2. Prüfen Sie das Ergebnis im Aufstellplan und im Sicht „Betriebsbild“.
  3. Mit Umläufe zurücksetzen heben Sie Zuordnungen wieder auf (Umläufe im Status „Wird durchgeführt“ bleiben zugeordnet).

Hinweise

  • Bei Wechsel der Ausrückphase wird die Disposition mit der aktuellen Hofbelegung neu berechnet.
  • Erforderliche Berechtigungen: Umlaufdisposition, UmlaufdispositionStarten, UmlaufdispositionLöschen.

Verwandte Themen

  • Umlaufdisposition  –  Einführung
  • Manuelle Fahrzeug-/Stellplatzdisposition
  • Dialog „Kursaufstellplan“
