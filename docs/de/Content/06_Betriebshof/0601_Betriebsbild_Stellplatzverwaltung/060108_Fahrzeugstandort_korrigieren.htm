Fahrzeugstandort korrigieren

Mit dem Dialog Standort setzen können Sie den Standort eines Fahrzeugs im Betriebshof manuell korrigieren, ein Fahrzeug auf einen Stellplatz verschieben oder aus dem Betriebshof entfernen.

Dialog öffnen

  • Symbolleiste Dialog → Standort setzen
  • Oder: Kontextmenü eines Fahrzeugs → Standort setzen…

Fahrzeugstandort korrigieren

  1. Wählen Sie das Fahrzeug in der Auswahlliste. Der Fahrzeugtyp wird neben dem Feld angezeigt.
  2. Wählen Sie den Betriebshof, in dessen Bereich das Fahrzeug platziert werden soll.
  3. Wählen Sie den Stellplatz im Feld Ort. Unterhalb wird die Spur­länge angezeigt (sofern mehrere Fahrzeuge dort stehen können).
  4. Geben Sie im Feld Position den Abstand in Metern vom Spur­anfang an. Das System schlägt automatisch eine optimale Position hinter den bereits abgestellten Fahrzeugen vor.
  5. Klicken Sie auf Übernehmen, um die Änderung zu speichern.
  6. Klicken Sie auf Schließen, um den Dialog zu beenden.

Fahrzeug aus Betriebshof entfernen

  1. Wählen Sie das Fahrzeug aus.
  2. Setzen Sie im Feld Betriebshof den Eintrag <Netz>.
  3. Bestätigen Sie mit Übernehmen → das Fahrzeug wird aus dem Betriebshof entfernt.
  4. Dialog mit Schließen beenden.

Besonderheiten

  • Wenn der Dialog über das Kontextmenü eines Fahrzeugs geöffnet wurde, ist das Fahrzeug bereits vorausgewählt.
  • Bei Auswahl eines Fahrzeugs werden aktuelle Daten (Ort, Position) automatisch geladen.
  • Wenn das Fahrzeug Teil einer Traktion ist, wird dies angezeigt. Über die Schaltfläche Trennen kann die Traktion aufgelöst werden, bevor der Standort korrigiert wird.
  • Die Spur­länge wird nur angezeigt, wenn dort mehrere Fahrzeuge erlaubt sind und es sich nicht um einen Stauraum handelt.
  • Bei Stellplätzen für nur ein Fahrzeug oder Stauraum ist keine Positionsangabe möglich.
  • Wird eine Position gewählt, an der kein Platz vorhanden ist, zeigt das System eine Warnung oder Fehlermeldung mit den blockierenden Fahrzeugen.

Verwandte Themen

  • Sicht „Betriebsbild“
  • Bedienhandlungen am Betriebsbild ausführen
  • Dialog "Stellplatz bearbeiten"
  • Traktionen – Kuppeln/Trennen
