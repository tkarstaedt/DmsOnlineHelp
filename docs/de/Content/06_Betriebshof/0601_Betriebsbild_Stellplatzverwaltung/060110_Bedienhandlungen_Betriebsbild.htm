Bedienhandlungen am Betriebsbild ausführen

Dieses Topic bündelt häufige Aufgaben im Bereich Betriebshof von PEAK.DMS. Jede Aufgabe enthält eine kurz gefasste Schrittfolge und verweist bei Bedarf auf Detail-Topics.

Fahrzeugsuche (Strg+F)

Mit der Fahrzeugsuche ermitteln Sie den Standort eines Fahrzeugs – auf dem Betriebshof oder auf der Strecke.

Aufrufen

  • Im Betriebsbild auf eine freie Fläche klicken → Kontextmenü Fahrzeugsuche…
  • Menü Bearbeiten → Fahrzeugsuche…
  • Werkzeugleiste → Schaltfläche Fahrzeugsuche
  • Tastenkombination Strg+F

Suchkriterien

  • Fahrzeugnummer
  • Traktion
  • Umlauf (es werden die Umläufe der aktuellen Ausrückphase angeboten, auf die ein Fahrzeug angemeldet ist)

Ergebnis

  • Fahrzeug steht im Betriebshof: Das zugehörige Betriebsbild wird geöffnet und das Fahrzeug kurzzeitig hervorgehoben (Blinken).
  • Fahrzeug ist im Einsatz: Es öffnet sich der Dialog Fahrzeugakte mit dem zuletzt vom RBL gemeldeten Standort.

Stellplatz sperren

  1. Öffnen Sie das Betriebsbild (siehe Sicht „Betriebsbild“) oder die Liste der Stellplätze (siehe Sicht „Stellplätze“).
  2. Stellplatz wählen → Bearbeiten… (öffnet den Dialog, siehe Dialog "Stellplatz bearbeiten").
  3. Auf dem Register Stellplatzdaten die Option Gesperrt aktivieren, Sperrgrund und optional Gültig von/bis setzen.
  4. OK wählen, um zu speichern.

Stellplatz entsperren

  1. Dialog Stellplatz bearbeiten öffnen (siehe Dialog "Stellplatz bearbeiten").
  2. Option Gesperrt deaktivieren und ggf. Zeitraum/Grund bereinigen.
  3. OK wählen.

Nutzungsart hinzufügen

  1. Stellplatz auswählen → Bearbeiten… (siehe Dialog "Stellplatz bearbeiten").
  2. Register Nutzungsarten öffnen → Hinzufügen….
  3. Nutzungsart wählen (z. B. Umlauf, Werkstatt, Versorgung, Reserve, Stauraum), optional Farbe/Zeitraum setzen.
  4. Mit OK bestätigen.

Nutzungsart entfernen

  1. Im Register Nutzungsarten die zu entfernende Zeile markieren.
  2. Entfernen wählen → Bestätigen → OK.

Fahrzeugtyp zulassen

  1. Stellplatz öffnen (siehe Dialog "Stellplatz bearbeiten").
  2. Register Fahrzeugtypen → Hinzufügen….
  3. Fahrzeugtyp (und ggf. Muster) wählen, optional Hinweis ergänzen.
  4. OK wählen.

Fahrzeugtyp entfernen

  1. Im Register Fahrzeugtypen die Zeile markieren.
  2. Entfernen wählen → OK.

Fahrerinformation verfassen

Fahrerinformationen dienen der gezielten Kommunikation (z. B. Hinweise zum Ausrücken, geänderte Hofwege). Die Verfügbarkeit ist projektspezifisch.

  1. Fahrauftragsliste öffnen (siehe Sicht „Fahraufträge“ anzeigen) oder ein Fahrzeug/Stellplatz im Betriebsbild auswählen (siehe Sicht „Betriebsbild“).
  2. Kontextmenü → Fahrerinformation verfassen… (Bezeichnung kann abweichen).
  3. Betreff und Nachricht eingeben, Zielgruppe/Empfänger festlegen, ggf. Gültigkeit/Bestätigungsanforderung setzen.
  4. Senden bzw. OK wählen.

Umsetzfahrt anlegen

Bewegen Sie ein Fahrzeug planbar zwischen Stellplätzen (Quelle → Ziel). Detaillierte Felder und Optionen finden Sie im eigenen Dialog-Topic.

  1. Aus Betriebsbild oder Liste starten (Kontextmenü) oder direkt über Aufträge (siehe Umsetzfahrt (Dialog) ).
  2. Fahrzeug, Quelle, Ziel, Start und optional Ende festlegen.
  3. Priorität und Grund setzen, Ausführungsart wählen → OK.

Hinweise

  • Rechte: Einige Aktionen erfordern Schreibrechte bzw. spezifische Rollen.
  • Projektspezifische Felder/Benennungen sind möglich; folgen Sie Ihrer Konfiguration.
  • E-Bus: Ladefenster, SOC und Ladezonen bei Zielwahl berücksichtigen.
