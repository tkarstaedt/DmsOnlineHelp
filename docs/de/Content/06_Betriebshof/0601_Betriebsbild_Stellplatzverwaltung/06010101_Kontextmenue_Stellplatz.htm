Kontextmenü „Stellplatz“ verwenden

Über das Kontextmenü eines Stellplatzes führen Sie in PEAK.DMS schnelle Pflege- und Navigationsaktionen aus, z. B. Sperren/Entsperren, Nutzungsarten oder zulässige Fahrzeugtypen pflegen sowie zur grafischen oder tabellarischen Ansicht springen.

Kontextmenü öffnen

  1. Klicken Sie im Betriebsbild einen Stellplatz mit der rechten Maustaste an.
  2. Wählen Sie die gewünschte Aktion aus der Liste.

Aktionen im Überblick

Die verfügbaren Einträge sind projektspezifisch. In der Standardkonfiguration stehen u. a. folgende Aktionen zur Verfügung:

Menüeintrag Kurzbeschreibung
Stellplatz bearbeiten… Öffnet den Dialog „Stellplatz bearbeiten“.
Sperren Setzt Gesperrt; optional Sperrgrund/Gültigkeit.
Entsperren Hebt eine bestehende Sperre auf.
Nutzungsart hinzufügen… Fügt dem Stellplatz eine Nutzung hinzu.
Nutzungsart entfernen Entfernt eine ausgewählte Nutzung.
Fahrzeugtyp zulassen… Ergänzt zulässige Fahrzeugtypen/-muster.
Fahrzeugtyp entfernen Entfernt Typ/Muster aus der Zulassung.
Im Betriebsbild anzeigen Zentriert und markiert den Stellplatz.
In „Stellplätze“ anzeigen Öffnet die Liste gefiltert auf den Stellplatz.
Umsetzfahrt anlegen… Erstellt eine Umsetzfahrt (falls freigeschaltet).
Historie Zeigt Änderungsprotokoll (falls vorhanden).

Typische Workflows

  1. Stellplatz sperren: Sperren wählen → Sperrgrund und Zeitraum setzen → OK.
  2. Nutzungsart pflegen: Nutzungsart hinzufügen… → Art wählen → optional Zeitraum → OK.
  3. Zulässige Fahrzeugtypen aktualisieren: Fahrzeugtyp zulassen… → Typ/Muster wählen → OK.
  4. Schneller Wechsel zur Liste: In „Stellplätze“ anzeigen → Datensatz prüfen/bearbeiten.

Berechtigungen und Hinweise

  • Einzelne Menüeinträge erfordern entsprechende Rechte.
  • Darstellung und Farblogik der Nutzungen werden im Betriebsbild visualisiert.
  • E-Bus-Projekte: Umsetzfahrten und Ladepunkt-Bezüge können zusätzliche Einträge aktivieren.
