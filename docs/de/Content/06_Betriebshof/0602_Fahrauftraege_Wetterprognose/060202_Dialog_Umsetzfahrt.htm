Dialog „Umsetzfahrt“ (Umsetzfahrt anlegen)

Im Dialog „Umsetzfahrt“ erstellen oder ändern Sie in PEAK.DMS eine geplante Fahrzeugbewegung innerhalb des Betriebshofs (von Stellplatz A nach Stellplatz B) inklusive Zeitpunkt, Priorität und Begründung.

Dialog öffnen

  1. Aus dem Betriebsbild: Stellplatz oder Fahrzeug rechtsklicken → Umsetzfahrt anlegen…. Für den Einstieg siehe Sicht „Betriebsbild“.
  2. Aus der Sicht „Stellplätze“: Stellplatz markieren → Kontextmenü > Umsetzfahrt anlegen…. Zur Listenarbeit siehe Sicht „Stellplätze“.
  3. Aus Fahraufträgen (falls freigeschaltet): Neuer Auftrag vom Typ „Umsetzfahrt“. Weitere Informationen finden Sie in Sicht „Fahraufträge“ anzeigen.

Voraussetzungen

  • Schreibrechte für Umsetzfahrten.
  • Quell- und Zielstellplatz existieren und sind nicht dauerhaft gesperrt.
  • Fahrzeug ist verfügbar (nicht in Werkstatt/Versorgung gebunden; projektspezifisch).

Felder und Optionen (Standard)

Die tatsächlichen Felder können projektspezifisch variieren. Typische Eingaben im Dialog „Umsetzfahrt“:

Feld Beschreibung
Fahrzeug Auswahl des zu bewegenden Fahrzeugs (Nummer/Typ).
Quelle (von) Aktueller Stellplatz bzw. Bereich.
Ziel (nach) Gewünschter Stellplatz bzw. Bereich.
Start Geplanter Beginn (Datum/Uhrzeit).
Ende (optional) Geplantes Ende/Deadline.
Priorität Dringlichkeit für Disposition/Abarbeitung.
Grund Kurzbegründung (z. B. Aufstellplan, Werkstatt, Reserve).
Hinweis (optional) Interner Kommentar für Leitstelle/Service.
Ausführungsart Manuell (Fahrer) oder intern (betrieblicher Prozess; projektspezifisch).

Umsetzfahrt anlegen

  1. Öffnen Sie den Dialog „Umsetzfahrt“ (siehe Dialog „Umsetzfahrt“ (Umsetzfahrt anlegen)) und wählen Sie Fahrzeug.
  2. Prüfen Sie Quelle (von) und wählen Sie Ziel (nach).
  3. Setzen Sie Start (und optional Ende) sowie Priorität.
  4. Erfassen Sie Grund und ggf. Hinweis.
  5. Wählen Sie die Ausführungsart und bestätigen Sie mit OK.

Umsetzfahrt bearbeiten oder stornieren

Bestehende Umsetzfahrten können angepasst oder aufgehoben werden.

  • Bearbeiten: Umsetzfahrt in der Liste/Anzeige wählen → Bearbeiten… → Felder anpassen → OK.
  • Stornieren: Umsetzfahrt wählen → Abbrechen/Stornieren → Grund angeben → OK.

Hinweise und Best Practices

  • Beachten Sie Stellplatz-Sperren und Nutzungsarten (Umlauf, Werkstatt, Versorgung, Reserve).
  • Prüfen Sie Konflikte mit Aufstell- oder Kursplanungen (projektspezifische Regeln).
  • E-Bus: Ziel möglichst ladefähig wählen; Ladefenster und SOC beachten.

Speichern und schließen

  • OK: Umsetzfahrt speichern und Dialog schließen.
  • Abbrechen: Änderungen verwerfen.
