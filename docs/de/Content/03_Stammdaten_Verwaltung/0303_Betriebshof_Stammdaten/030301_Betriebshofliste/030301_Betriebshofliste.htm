Die Sicht "Betriebshöfe"

Die tabellarische Sicht "Betriebshöfe " listet alle im System konfigurierten Depots auf und zeigt je Depot den Fahrzeugbestand (Soll/Ist) sowie weitere Kennzahlen. Von hier gelangen Sie direkt zum Dialog Betriebshofinformation zur Pflege hofbezogener Parameter.

Funktionen & Einstellungen

Aus der Betriebshofliste heraus können verschiedene Aktionen ausgeführt werden:

  • Betriebshof bearbeiten: Durch einen Doppelklick auf eine Zeile oder über das Kontextmenü kann der Dialog "Betriebshofinformation" geöffnet werden.
  • Fahrerinformation verfassen: Im Dialog "Betriebshofinformation" können Lauftextinformationen für das Fahrerterminal des jeweiligen Betriebshofs verfasst werden.
  • Betriebsbild öffnen: Über das Kontextmenü kann direkt zum grafischen Betriebsbild des ausgewählten Betriebshofs navigiert werden.
