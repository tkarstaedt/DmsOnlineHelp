Fenstermanagement – Grundlagen

Das Fenstermanagement in PEAK.DMS ermöglicht es Ihnen, die Arbeitsoberfläche flexibel an Ihre Bedürfnisse anzupassen. Sie können neue Anwendungsfenster öffnen, den Anzeigebereich gliedern und Unterfenster nutzen. Ihre Einstellungen lassen sich im Benutzerprofil speichern, sodass Sie beim nächsten Anmelden sofort in Ihrer gewohnten Arbeitsumgebung weiterarbeiten.

Neues Anwendungsfenster öffnen

Sie können mehrere Instanzen des Anwendungsfensters parallel öffnen, z. B. um verschiedene Arbeitsaufgaben gleichzeitig zu bearbeiten:

  • Öffnen Sie im Menü Datei den Eintrag Neues Anwendungsfenster .
  • Ein weiteres Anwendungsfenster wird gestartet, das Sie unabhängig nutzen können.

Anzeigebereich gliedern

Der Anzeigebereich kann in mehrere Unterfenster aufgeteilt werden. Dies ermöglicht die gleichzeitige Anzeige unterschiedlicher Sichten:

  • Vertikal teilen: Unterfenster werden nebeneinander dargestellt.
  • Horizontal teilen: Unterfenster werden untereinander angezeigt.

Unterfenster nutzen

Unterfenster enthalten jeweils eine Sicht. Sie können darin Daten betrachten, vergleichen oder parallel bearbeiten.

  • Ziehen Sie eine Registerkarte mit der Maus in den gewünschten Bereich, um ein neues Unterfenster zu erzeugen.
  • Verschieben Sie Sichten per Drag-and-drop zwischen Unterfenstern.
  • Schließen oder maximieren Sie Unterfenster nach Bedarf.

Speichern im Benutzerprofil

Ihr aktuelles Fensterlayout können Sie im Benutzerprofil speichern. Beim nächsten Anmelden wird es automatisch wiederhergestellt.

  1. Wählen Sie im Menü Datei > Benutzerprofil speichern .
  2. Geben Sie einen Profilnamen ein oder überschreiben Sie ein bestehendes Profil.
  3. Klicken Sie auf OK , um das Layout zu sichern.
