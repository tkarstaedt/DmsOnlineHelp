Datum und Uhrzeit auswählen

In vielen Dialogen von PEAK.DMS geben Sie ein Datum oder eine Uhrzeit an, um den zeitlichen Bezug einer Aktion oder eines Objekts festzulegen. Sie können Werte direkt eingeben oder über Auswahlfelder bestimmen.

Verweis: Grundlegende Hinweise zum Arbeiten mit Dialogen finden Sie in Dialogaufbau verstehen . Tipps zur Eingabe in Feldern stehen in Bedienelemente nutzen .

Datum auswählen

  • Klicken Sie in das Datumsfeld . Ein Kalender wird eingeblendet.
  • Wählen Sie das gewünschte Datum per Mausklick aus.
  • Blättern Sie mit den Pfeilen am oberen Rand zwischen Monaten oder Jahren.
  • Alternativ: Geben Sie das Datum manuell ein (Format: TT.MM.JJJJ).

Uhrzeit eingeben

  • Klicken Sie in das Uhrzeitfeld .
  • Geben Sie die Uhrzeit manuell ein (Format: hh:mm).
  • Nutzen Sie Pfeilschaltflächen oder das Mausrad, um Stunden und Minuten schrittweise zu ändern.

Kombination von Datum und Uhrzeit

In einigen Dialogen wählen Sie sowohl Datum als auch Uhrzeit, z. B. beim Planen eines Umlaufs. Achten Sie darauf, beide Felder korrekt auszufüllen, da sie gemeinsam gespeichert werden.

Tastatursteuerung

  • Tab : Wechsel zum nächsten Feld.
  • Pfeiltasten : Navigation innerhalb des eingeblendeten Kalenders.
  • Enter : Übernahme des ausgewählten Werts.

Fehler vermeiden

  • Stellen Sie sicher, dass das Datum im zulässigen Bereich liegt (z. B. keine Vergangenheit bei Planungen).
  • Geben Sie Uhrzeiten im 24-Stunden-Format an.
  • Bei fehlerhaften Eingaben zeigt das System eine Meldung an (→ Eingaben prüfen und Feedback verstehen ).
