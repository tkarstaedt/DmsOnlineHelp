Benutzerprofil erstellen

Mit einem Benutzerprofil speichern Sie Ihre persönliche Arbeitsumgebung in PEAK.DMS . Dazu richten Sie Fensterlayout und Sichten ein und sichern diese Einstellungen als Profil, das Sie später schnell laden können.

Arbeitsumgebung vorbereiten

  • Öffnen Sie die Sichten, die Sie regelmäßig benötigen.
  • Teilen Sie den Anzeigebereich nach Bedarf in Unterfenster und ordnen Sie die Registerkarten.
  • Passen Sie Spalten und Symbolleisten an, sodass häufig genutzte Funktionen schnell erreichbar sind.

Profil speichern

  1. Öffnen Sie die Menüleiste und wählen Sie Datei .
  2. Klicken Sie auf Benutzerprofil speichern .
  3. Geben Sie im Dialog einen Profilnamen ein.
  4. Optional: Fügen Sie eine Beschreibung hinzu (Zweck, Rolle, Datum).
  5. Klicken Sie auf OK , um das Profil zu erstellen.
