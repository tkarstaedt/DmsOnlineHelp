Benutzerprofile verwalten

In der Profilverwaltung bearbeiten Sie eigene Profile und, bei entsprechender Berechtigung, auch Profile anderer Benutzer. Profile können als Vorlage dienen und sichern Ihr Fenster‑ und Sichtenlayout für wiederkehrende Aufgaben.

Dialog "Profilverwaltung" öffnen

  • Klicken Sie im Menü Datei  →  Profile verwalten
  • Oder: Klicken Sie in der Symbolleiste Datei  →  3‑Punkt‑Schaltfläche rechts neben dem Diskettensymbol
  • Alternativ: Klicken Sie im Menü Fenster  →  Profile verwalten

Bereiche der Profilverwaltung

  • Globale Profile: Zentrale Profile, für alle Benutzer sichtbar (nur Administratoren dürfen anlegen/ändern/löschen).
  • Persönliche Profile: Nur für den jeweiligen Besitzer sichtbar und vollständig von ihm steuerbar (Administrator kann bei Bedarf zugreifen).
  • Profile anderer Benutzer anzeigen: Optionaler Bereich, der zusätzliche Rechte erfordert.

Was speichert ein Profil?

  • Größe/Position des Anwendungsfensters (inkl. Mehrmonitorbetrieb).
  • Welche Datensichten/Dokumente geöffnet sind und deren Anordnung.
  • Filter, Sortierungen, Spalteneinstellungen und benutzerdefinierte Kartentitel in Sichten.

Was speichert ein Profil nicht?

  • Einstellungen generischer Sichten (Konfiguration/Parameter/Prozessdaten).
  • Menüeinstellungen (gelten benutzerweit unabhängig vom Profil).

Aktionen

Profil laden

  1. Klicken Sie auf Profil laden.
  2. Wählen Sie ein Profil aus; die aktuell geöffneten Sichten werden geschlossen und das gewählte Profil geladen.

Profil speichern

  • Menü Fenster  →  Profil speichern  –  bestätigen Sie das Überschreiben des geöffneten Profils.
  • Oder Symbolleiste: Profil speichern (gleiche Wirkung).

Profil erstellen/ändern/löschen

  • Erstellen: Neues Profil anlegen und aktuelles Layout speichern.
  • Ändern: Layout anpassen und Profil speichern.
  • Löschen: Nicht mehr benötigte Profile entfernen (global nur durch Administrator).

Hinweise & Rechte

  • Nur mit entsprechenden Rechten können Sie fremde Profile einsehen oder bearbeiten.
  • Globale Profile sind lesbar für alle, änderbar nur durch Administratoren.

SSB – Anmeldung und Profilwahl

  • Nach SSO‑Anmeldung können Sie am Begrüßungsbildschirm das letzte oder ein gespeichertes Benutzerprofil wählen; Profile steuern u. a. geöffnete Sichten, Anordnung und Filter.
  • SSO‑Parameter (Sitzungsdauer/Timeout/Lockout) richten sich nach der SSB‑Policy.
