Benutzerprofile effizient nutzen

Mit Benutzerprofilen richten Sie Ihre Arbeitsumgebung in PEAK.DMS passgenau ein und wechseln bei Bedarf schnell zwischen unterschiedlichen Aufgaben. Die folgenden Empfehlungen helfen Ihnen, Profile strukturiert anzulegen, zügig zu wechseln und dauerhaft übersichtlich zu halten.

Profilvarianten einsetzen

  • Rollenprofile nutzen: Legen Sie Profile für typische Rollen an (z. B. Leitstelle, Werkstatt, Planung), um passende Sichten und Layouts vorzudefinieren.
  • Aufgabenprofile trennen: Erstellen Sie Profile für wiederkehrende Aufgaben (z. B. Monitoring, Analyse), um Kontextwechsel zu beschleunigen.
  • Naming-Konventionen einhalten: Verwenden Sie sprechende Namen und optional Versionszusätze (z. B. „… kompakt“, „… v2“).

Schnell zwischen Profilen wechseln

  • Beim Anmelden auswählen: Wählen Sie das Zielprofil im Bereich Ihr Profil , um ohne weitere Schritte in der richtigen Umgebung zu starten.
  • Im Betrieb umschalten: Wechseln Sie Profile über das Menü Datei , wenn sich Ihr Fokus ändert (z. B. von Disposition auf Analyse).
  • Aktuelles Layout sichern: Speichern Sie vor dem Wechsel Ihr Layout, damit Ihre Anpassungen nicht verloren gehen.

Profile pflegen

  • Regelmäßig aktualisieren: Passen Sie Profile an neue Arbeitsweisen, Sichten oder Symbolleisten an.
  • Altlasten entfernen: Löschen Sie veraltete oder selten genutzte Profile, um die Auswahlliste schlank zu halten.
  • Varianten konsolidieren: Fassen Sie sehr ähnliche Profile zusammen, um Doppelpflege zu vermeiden.

Layout optimal vorbereiten

  • Sichten anordnen: Nutzen Sie Unterfenster und Registerkarten, um häufig kombinierte Sichten nebeneinander bereitzustellen.
  • Spalten anpassen: Blenden Sie relevante Spalten ein, ordnen Sie sie neu und speichern Sie das Layout im Profil.
  • Symbolleisten kuratieren: Platzieren Sie die wichtigsten Aktionen sichtbar, um Klickwege zu verkürzen.

Häufige Probleme vermeiden

  • Ungesicherte Änderungen: Speichern Sie das Profil nach Layout-Anpassungen, bevor Sie das Profil wechseln.
  • Konflikte beim Löschen: Ein geladenes Profil kann nicht gelöscht werden. Laden Sie zuerst ein anderes Profil.
  • Unklare Profilnamen: Versehen Sie Profile mit Zweck und Zielgruppe, um Verwechslungen zu verhindern.

Checkliste: Do’s & Don’ts

Do’s Don’ts
Profile nach Rolle und Aufgabe strukturieren. Beliebige Namen ohne Kontext vergeben.
Layout-Änderungen sofort im Profil speichern. Mit unsicherem Layout das Profil wechseln.
Regelmäßig aufräumen und konsolidieren. Veraltete Profile jahrelang mitführen.
Symbolleisten und Sichten auf die Aufgabe zuschneiden. Überladenes Layout ohne Priorisierung nutzen.
