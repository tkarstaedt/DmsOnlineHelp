# 7 Werkstatt und Instandhaltung

Das Modul **„Werkstatt“** dient der digitalen Steuerung von Wartungs- und Reparaturaufträgen. Es ermöglicht dem Werkstattpersonal, offene Arbeiten einzusehen und deren Erledigung direkt am Fahrzeug zurückzumelden.

## 7.1 Werkstattaufträge einsehen
Tippen Sie auf dem Dashboard auf die Kachel **„Werkstatt“** (Fahrzeug-Symbol). Es erscheint eine Liste aller Aufträge, gruppiert nach Fahrzeugnummern.

### Der Auftragsstatus
Jeder Eintrag ist mit einer farbigen Markierung versehen, die den aktuellen Bearbeitungsstand anzeigt:

*   **Geplant** (Orange/Rot): Der Auftrag ist im System vorgemerkt, wurde aber noch nicht begonnen.
*   **In Bearbeitung** (Rot): Die Arbeit an diesem Auftrag hat begonnen.
*   **Erledigt** (Hellgrün): Die Arbeiten wurden erfolgreich abgeschlossen.

Zusätzlich sehen Sie wichtige Details wie den Arbeitsplatz (z. B. „FBM“), den geplanten Zeitraum und eventuelle Bemerkungen der Disposition oder des Fahrers.

## 7.2 Aufträge fertig melden
Wenn Sie eine Reparatur oder Wartung abgeschlossen haben, müssen Sie dies im System dokumentieren, damit das Fahrzeug wieder für den Betrieb freigegeben werden kann.

1.  Suchen Sie in der Werkstattliste das betreffende Fahrzeug und tippen Sie auf den Auftrag (z. B. einen Eintrag mit Status *„In Bearbeitung“*).
2.  Die **Detailansicht** öffnet sich. Hier können Sie die Auftragsdaten (Ersteller, Text, Reservierung) noch einmal überprüfen.
3.  Tippen Sie am unteren Bildschirmrand auf die rote Schaltfläche (z. B. **„Fertig melden“** oder **„Abschließen“**).
4.  Ein Bestätigungsfenster erscheint: „Auftrag abschließen?“.
5.  Bestätigen Sie mit **„Ja“**.

*Ergebnis:* Sie kehren automatisch zur Liste zurück. Der Status des Auftrags ändert sich sofort auf *„Erledigt“* (Grün). Das Fahrzeug steht der Disposition nun wieder zur Verfügung (sofern keine weiteren sperrenden Aufträge vorliegen).
