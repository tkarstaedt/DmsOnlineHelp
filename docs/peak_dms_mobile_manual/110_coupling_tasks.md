# 11 Kuppelaufträge

Das Modul **„Kuppelaufträge“** dient der Steuerung von Fahrzeugtraktionen. Hier erhalten Rangierer und Fahrer Anweisungen, welche Fahrzeuge zu einem Zugverband zusammengestellt (gekuppelt) oder getrennt (entkuppelt) werden sollen.

## 11.1 Kuppelaufträge einsehen
Tippen Sie auf dem Dashboard auf die Kachel **„Kuppelaufträge“** (Symbol mit zwei verbundenen Kettengliedern).

### Die Auftragsliste
Die Liste zeigt alle offenen Rangieraufträge für das gewählte Depot. Jeder Eintrag informiert Sie über:
*   **Vorgangsart:** Ein Symbol zeigt an, ob gekuppelt (Pfeile zueinander) oder entkuppelt (Pfeile voneinander) werden soll.
*   **Fahrzeuge:** Die Nummern der beteiligten Fahrzeuge (z. B. `4101` + `4102`).
*   **Ort:** Das Gleis, auf dem der Vorgang stattfindet.

## 11.2 Kupplungsvorgänge bestätigen
Nachdem Sie die physische Kupplung oder Trennung der Fahrzeuge durchgeführt haben, müssen Sie den Vorgang im System bestätigen. Dies aktualisiert automatisch die Fahrzeugverbände in der Disposition.

1.  Wählen Sie den entsprechenden Auftrag aus der Liste aus.
2.  In der Detailansicht sehen Sie die Zielkonfiguration (z. B. „Fahrzeug 4101 vorne, 4102 hinten“).
3.  Prüfen Sie, ob die Reihung der Fahrzeuge mit der Realität übereinstimmt.
4.  Tippen Sie auf die Schaltfläche **„Bestätigen“** (bzw. **„Fertig melden“**).
5.  Bestätigen Sie die Sicherheitsabfrage mit **„Ja“**.

*Ergebnis:* Der Auftrag verschwindet aus der Liste. In der Fahrzeugübersicht und der Betriebshofansicht werden die Fahrzeuge nun als eine Einheit (Traktion) oder als Einzelfahrzeuge dargestellt.
