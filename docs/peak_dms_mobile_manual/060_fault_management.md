# 6 Störungsmanagement

Über die Kachel **„Fahrzeugstörungen“** (Bus-Symbol mit Ausrufezeichen) gelangen Sie in den Bereich zur Erfassung und Verwaltung technischer Defekte. Dies ist essenziell, um die Einsatzfähigkeit der Flotte zu gewährleisten und die Werkstatt über Mängel zu informieren.

## 6.1 Störungsübersicht verstehen
Die Liste der Störungen zeigt Ihnen auf einen Blick, welche Fahrzeuge Probleme haben. Jeder Eintrag ist mit einem farbigen Code markiert, der die Schwere des Fehlers und die Auswirkung auf den Einsatz anzeigt:

*   **A – Kein Einsatz** (Rot): Das Fahrzeug ist nicht fahrbereit und darf nicht ausrücken (z. B. Bremsdefekt).
*   **B – Nur manueller Einsatz** (Rot): Das Fahrzeug darf nur unter bestimmten Auflagen oder durch manuelle Disposition eingesetzt werden.
*   **C – Bedingter Einsatz** (Schwarz/Weiß): Das Fahrzeug ist einsatzbereit, hat aber Einschränkungen (z. B. defekte Klimaanlage).
*   **D – Einsatz** (Grün/Oliv): Ein kleinerer Mangel, der den Betrieb nicht gefährdet (z. B. Kratzer im Lack).

Zusätzlich sehen Sie den Status der Bearbeitung (z. B. *„Gemeldet“* oder *„Erledigt“*).

## 6.2 Neue Störung erfassen
Wenn Sie einen Defekt am Fahrzeug feststellen, können Sie diesen direkt in der App melden.

1.  Tippen Sie in der Störungsliste auf das **Plus-Symbol (+)** (oder wählen Sie „Fahrzeugstörung anlegen“ im Menü).
2.  Das Formular „Fahrzeugstörung erstellen“ öffnet sich.
3.  Füllen Sie die Felder nacheinander aus:
    *   **Fahrzeug:** Geben Sie die Fahrzeugnummer ein (z. B. `1401`).
    *   **Störungsgrad auswählen:** Legen Sie die Priorität fest (A, B, C oder D).
    *   **Kategorie auswählen:** Tippen Sie auf das Feld, um den Katalog zu öffnen. Navigieren Sie durch die Baumstruktur (z. B. `EV2010, Fahrzeugbeleuchtung` > `0010, Scheinwerfer defekt`), bis Sie das passende Bauteil gefunden haben.
    *   **Meldung:** Das System trägt oft automatisch einen Text basierend auf der Kategorie ein. Sie können diesen ergänzen.
    *   **Empfehlung:** Geben Sie optional einen Handlungshinweis für die Werkstatt ein (z. B. „Sofort Ersatz bestellen“).
4.  Tippen Sie auf die rote Schaltfläche **„Erstellen“**.

*Ergebnis:* Die Störung wird gespeichert, an das DMS übertragen und erscheint sofort in der Übersicht mit dem entsprechenden Status-Code.

## 6.3 Störungen bearbeiten und abschließen
Um Details zu einer bestehenden Meldung anzusehen oder deren Status zu ändern:

1.  Tippen Sie in der Liste auf die betreffende Störung.
2.  In der Detailansicht sehen Sie die Historie und technische Details.
3.  Je nach Ihren Berechtigungen können Sie:
    *   Die Störung kommentieren.
    *   Den Status auf **„Erledigt“** setzen, wenn der Mangel behoben wurde.
