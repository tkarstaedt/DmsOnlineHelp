# 3 Die Benutzeroberfläche

Nach der Anmeldung gelangen Sie automatisch auf das **Dashboard** (die Startseite). Es dient als zentrale Steuerzentrale, von der aus Sie alle Funktionen der App erreichen.

## 3.1 Das Dashboard verstehen
Das Dashboard ist in einem Kachel-Layout (Grid) aufgebaut. Es bietet Ihnen direkten Zugriff auf die operativen Module. Je nach Ihren Benutzerrechten und der Konfiguration Ihres Unternehmens sehen Sie hier bis zu zehn verschiedene Funktionsbereiche.

Die wichtigsten Kacheln sind:

*   **Bewegungslisten:** Die Kacheln **„Einrückliste“** und **„Ausrückliste“** zeigen geplante Fahrten an.
*   **Fahrzeugstörungen:** (Bus-Symbol mit Ausrufezeichen) Hier erfassen und bearbeiten Sie technische Defekte.
*   **Meldungen:** (Kalender-Symbol) Hier finden Sie betriebliche Nachrichten und Hinweise.
*   **Betriebshofansicht:** Eine grafische Karte der aktuellen Fahrzeugpositionen im Depot.
*   **Operative Bereiche:** Kacheln für **„Werkstatt“**, **„Versorgung“** (Reinigung/Tanken) und **„VSP“** (Versorgungsspur-Steuerung).
*   **Fahrzeuge:** Der Zugriff auf die detaillierte Fahrzeugakte.

## 3.2 Den Dienststatus prüfen
Zwischen den Funktionskacheln finden Sie das Widget **„Dein aktueller Dienst“**. Dieses Bereich ist besonders für das Fahrpersonal relevant.

*   **Statusanzeige:** Hier sehen Sie Informationen zu Ihrem nächsten geplanten Dienst (z. B. „Status des Dienstantritts ist unbekannt“).
*   **Aktion:** Tippen Sie auf die dunkle Schaltfläche **„Dienstantritt bestätigen“**, um der Leitstelle zu signalisieren, dass Sie einsatzbereit sind.

## 3.3 Die Navigation nutzen
Die App folgt einem einfachen Navigationskonzept, das für die Einhandbedienung optimiert ist.

### Module öffnen
Tippen Sie einfach auf die entsprechende Kachel (z. B. **„Fahrzeuge“**), um in den jeweiligen Bereich zu wechseln.

### Zurück zur Übersicht
Innerhalb eines Moduls finden Sie am oberen linken Bildschirmrand einen **Pfeil nach links** (<). Tippen Sie darauf, um einen Schritt zurückzugehen oder zum Dashboard zurückzukehren.

### Benutzermenü und Logout
Oben links auf dem Dashboard (oder in Untermenüs) finden Sie ein **Profil-Icon** (Männchen-Symbol oder Ihr Benutzerkürzel).
Tippen Sie auf dieses Icon, um das Benutzermenü zu öffnen. Hier können Sie:
*   Die **„Einstellungen“** aufrufen.
*   Sich über **„Ausloggen“** vom System abmelden.
