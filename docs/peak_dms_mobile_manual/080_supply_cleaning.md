# 8 Versorgung und Reinigung

Das Modul **„Versorgung“** unterstützt Sie bei der täglichen Routinenarbeit auf dem Betriebshof. Hierzu zählen Tätigkeiten wie Innen- und Außenreinigung, Betankung, Besanden oder das Prüfen von Betriebsstoffen.

## 8.1 Versorgungsaufträge durchführen
Tippen Sie auf dem Dashboard auf die Kachel **„Versorgung“** (Icon mit Wassertropfen/Duschkopf).

### Die Auftragsliste
Sie sehen eine Liste aller offenen Aufgaben, sortiert nach Fahrzeugen oder Fälligkeit. Typische Aufträge sind:
*   *„Besanden (3 tägig)“*
*   *„Außenwäsche“*
*   *„Luftdruck prüfen“*

Wählen Sie den gewünschten Auftrag aus der Liste aus, um die Detailansicht zu öffnen.

## 8.2 Auftragsdauer erfassen
Im Gegensatz zu Werkstattaufträgen erfordert das System bei Versorgungsleistungen häufig die Erfassung der benötigten Arbeitszeit.

1.  Tippen Sie in der Detailansicht des Auftrags auf **„Fertig melden“**.
2.  Ein Dialogfenster **„Auftrag schließen“** erscheint.
3.  Tippen Sie in das Feld **„Dauer in Minuten“** und geben Sie die benötigte Zeit ein (z. B. `24`).
    *   *Hinweis:* Diese Angabe ist wichtig für die statistische Auswertung und Prozessoptimierung.
4.  Bestätigen Sie mit **„Ok“**.

*Ergebnis:* Der Auftrag wird als erledigt markiert (grüner Haken) und aus der Liste der offenen Aufgaben entfernt.

## 8.3 Schnellerfassung nutzen (VSP)
Wenn Sie an der Versorgungsspur (VSP) arbeiten und mehrere Aufgaben für ein Fahrzeug gleichzeitig erledigen (z. B. Tanken und Besanden), können Sie die Schnellerfassung nutzen.

1.  Tippen Sie auf dem Dashboard auf **„VSP“** oder scannen Sie den QR-Code des Fahrzeugs.
2.  Geben Sie die Fahrzeugnummer ein (z. B. `1401`).
3.  Die App zeigt alle für dieses Fahrzeug anstehenden Versorgungsaufgaben in einer Übersicht.
4.  Tippen Sie auf die Schaltfläche **„Alle Fertig melden“** (Checkmark-Symbol), um sämtliche gelisteten Aufgaben in einem Schritt abzuschließen.
