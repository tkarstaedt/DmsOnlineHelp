# 9 Betriebshofansicht (Grafische Disposition)

Die **Betriebshofansicht** ist ein visuelles Werkzeug, das Ihnen die aktuelle Belegung der Gleise und Abstellflächen in Echtzeit anzeigt. Sie dient der schnellen Orientierung auf dem Betriebsgelände und hilft Disponenten sowie Rangierern, freie Kapazitäten oder blockierte Fahrzeuge zu erkennen.

## 9.1 Die Karte bedienen
Tippen Sie auf dem Dashboard auf die Kachel **„Betriebshofansicht“** (Symbol mit versetzten Quadraten). Die Karte lädt die aktuelle Ansicht des Hauptdepots.

### Navigation in der Karte
Da Betriebshöfe oft weitläufig sind, wird meist nur ein Ausschnitt angezeigt. Sie können die Ansicht wie folgt steuern:

*   **Ausschnitt verschieben:** Wischen Sie mit einem Finger über den Bildschirm, um die Karte in die gewünschte Richtung zu verschieben (Panning).
*   **Vergrößern/Verkleinern:** Ziehen Sie zwei Finger auf dem Bildschirm auseinander, um hineinzuzoomen, oder ziehen Sie sie zusammen, um herauszuzoomen (Pinch-to-Zoom). So können Sie Details einzelner Gleise betrachten oder sich einen Gesamtüberblick verschaffen.

### Depot wechseln
Verfügt Ihr Verkehrsbetrieb über mehrere Standorte, finden Sie am unteren Bildschirmrand eine Leiste mit den Kürzeln der Depots (z. B. `FBM`, `FSE`, `FBW`).
*   Tippen Sie auf das entsprechende Kürzel, um die Karte des gewählten Betriebshofs zu laden.

## 9.2 Fahrzeugstatus in der Karte erkennen
Fahrzeuge werden in der grafischen Ansicht als rechteckige Blöcke auf den Gleisen dargestellt. In jedem Block steht die **Fahrzeugnummer**.

Die Farbe des Blocks signalisiert den aktuellen Status des Fahrzeugs im System:

*   **Grün/Weiß:** Das Fahrzeug ist betriebsbereit und hat keine einschränkenden Meldungen.
*   **Rot:** Es liegt eine gravierende Störung vor (Status A oder B) oder das Fahrzeug ist für den Auslauf gesperrt.
*   **Magenta/Violett:** Das Fahrzeug befindet sich in einem Werkstattprozess oder hat einen offenen Werkstattauftrag.
*   **Orange/Gelb:** Das Fahrzeug ist für eine Versorgung eingeplant oder hat einen bedingten Status.

> **Tipp:** Nutzen Sie diese Ansicht vor der Disposition, um sicherzustellen, dass ein geplantes Zielgleis tatsächlich physisch frei ist und genügend Restlänge für das Fahrzeug bietet.
