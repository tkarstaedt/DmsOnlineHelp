      Export & Documentation

      This topic describes how to data regarding coupling and decoupling procedures for , Audits, and Documentation – including data scope, formats, steps, and recommendations for reproducibility.

      Data Scope

      • : ID, Type (Couple/Decouple), Status, Timestamps Created/Confirmed/Canceled, Source (manual/automatic/Import), Vehicles (A/B), Traction, Depot/Track, Block/Duty, Comment.
      • Signaling: Traction Status (exists/resolved), Time, Source, Vehicles, optional Location.
      • Conflict/Check Events: Compatibility, Blocks, Supply Slots, Check Result.
      • Follow-up Effects (optional): Set flags Do not couple/Do not decouple, duration, reason for lifting.

      Formats

      • CSV – Universal, for BI/Excel; separator according to project settings.
      • Excel (XLSX) – For expert review and ad‑hoc pivot tables.
      • PDF – Signable proof (key date reports).

      Procedure (Export)

      1. Open Create Statistics and select the Coupling Management schema.
      2. Set filters (Time Period, Type, Source, Depot/Track, Vehicle Class, Line/Block) and confirm the selection.
      3. Check the preview and select → Export Statistics (CSV/Excel/PDF).
      4. Assign File Name, Key Date, and optional Report Profile (reusable filters).
      5. Save and document the storage location (Share/Archive).

      Documentation & Reproducibility

      • Save Report Profile (filters/columns) → Identical evaluation upon repetition.
      • Document Key Date (time period, time zone, version of rules/compatibility matrix).
      • Specify Sources (Signaling vs. Orders) and log deviations.
      • Storage: Securely store files for audit trail (path/share, permissions).

      Best Practices

      • Report median/percentiles for time-related ; treat outliers separately.
      • Report conflict classes separately (Compatibility, Block, Slot).
      • For audits, additionally generate a PDF with a Management Summary.
      • Data Minimization: only necessary fields; anonymize personal content.
