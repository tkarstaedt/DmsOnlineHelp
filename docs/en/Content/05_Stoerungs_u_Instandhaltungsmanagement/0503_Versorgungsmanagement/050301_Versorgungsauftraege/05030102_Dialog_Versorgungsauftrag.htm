      The "Supply Order" Dialog

      The "Supply Order" dialog can be used to create new and edit existing ones. Supply orders are tasks to be carried out on the vehicle (e.g., inspection, sanding) that, depending on the configuration, block the vehicle for for a specific period.

      Accessing the Dialog

      You can open the dialog in the following ways:

      • Create New Order:
        • Click the Vehicle Supply button in the "Dialog" toolbar.
        • Open the in the "Supply Orders" and select Add… .
      • Edit Existing Order:
        • Double-click an entry in the "Supply Orders" .
        • Open the of an order in the "Supply Orders" and select Edit… .

      Description of the Dialog Fields

      "Basic Data" Tab

      Figure 1: Vehicle Supply Dialog

      Field Description
      Vehicle Selection of the vehicle for which the order applies.
      Supply Type Type of the (e.g., washing, interior cleaning, inspection).
      Remark Optional comment on the order.
      Start Planned start of the order (date and time).
      End The planned end is automatically calculated based on the stored duration for the supply type. It can be manually adjusted if required.
      Depot The where the supply is to take place.
      Processing

      This section is only active in edit mode and with the corresponding permission:

      • Completed: Marks the order as reported finished.
      • Cancel: Cancels the order without successful execution.
