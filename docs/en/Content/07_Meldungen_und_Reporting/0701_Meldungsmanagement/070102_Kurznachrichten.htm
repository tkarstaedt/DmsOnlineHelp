      View and create short messages

      Short messages enable fast communication between the Control Center and driving staff. Unlike system messages (which are generated automatically), short messages are written manually and addressed to specific recipients. They can contain notes, instructions, or questions and are displayed in the driver terminal or the Control Center view.

      Function and benefits

      • Purpose: Exchange of short text messages for coordination during operations (e.g., changed departure time, additional task).
      • Distinction: Short messages are personal communications and are managed separately from system messages and driving orders.
      • Direction: Messages can be sent from dispatchers to drivers or from drivers to the Control Center (project‑dependent).
      • Documentation: All short messages are logged in the system and can be viewed in the operational logbook.

      Input fields (default)

      Field Description
      Sender The person or role composing the message.
      Recipient Individual driver or group (e.g., all drivers on a line).
      Priority Importance level (e.g., normal, high).
      Subject Short headline of the message.
      Message text Free‑text field for your message.
      Acknowledgement required Indicates whether the recipient must acknowledge receipt.
      Attachment (optional) File or link; project‑dependent.

      Create a short message

      1. Open the Short messages module from the navigation tree or via the context menu in the depot view.
      2. Click New or Create message.
      3. Select Recipient and, if applicable, a group. Set the Priority.
      4. Enter Subject and Message text. Add an attachment if needed.
      5. Activate Acknowledgement required if the recipient must confirm receipt.
      6. Click Send. The message appears in the recipientâ€™s inbox.

      View short messages

      • Select the Short messages view. Unanswered messages are highlighted.
      • Double‑click a message to open the detail dialog with full text, attachments, and receipt status.
      • Acknowledge receipt if requested. If you want to reply, enter a text and click Send.
      • Use filters by date, priority, or status for better overview.

      Notes

      • Short messages can be time‑limited depending on the project (e.g., display only during the duty).
      • In the depot view, new messages are often displayed as a pop‑up or via a marker on the vehicle icon.
      • All sent messages are logged in the operational logbook (see there).
