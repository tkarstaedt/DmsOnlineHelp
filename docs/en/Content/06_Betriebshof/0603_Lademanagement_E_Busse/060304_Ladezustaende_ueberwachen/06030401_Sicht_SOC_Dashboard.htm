      Display view "SOC-Dashboard"

      The SOC dashboard (State of Charge) provides current charging information for each electric vehicle. The view helps identify vehicles with critical SOC, estimate ranges, and plan suitable charging orders. This section explains the dashboardâ€™s columns, filters, and typical actions.

      Columns (default configuration)

      The tabular view contains the dynamic fields from the electrical process data of the vehicle dossier, for example:

      Column Description
      Vehicle Number and vehicle type.
      Current SOC [%] State of charge of the traction battery in percent.
      Current SOC [kWh] Currently usable energy of the battery.
      Remaining range without block Remaining range in km without considering scheduled trips.
      Remaining range with block Remaining range with respect to the currently dispatched block.
      Connected Indicates whether the vehicle is connected to a charging point and, if applicable, which charging phase is active (charging, preconditioning, balancing).
      Estimated charge completion Forecast time at which the battery reaches the target SOC (full charge).
      Alarms Displays warnings or notices when thresholds are undershot.

      Filtering and sorting

      Use filters and sorting to focus on critical vehicles:

      • Filter by SOC: Show only vehicles with SOC below a specified value (e.g., < 40%).
      • Sort by remaining range: Vehicles with the lowest remaining range appear at the top.
      • Filter by connected: Show only vehicles that are currently charging or at a charging point.

      Actions

      • Create charging order: Directly from the dashboard, you can create a charging order via right-click (see Create Charging Order).
      • Plan transfer move: If a vehicle is to be moved to another area, you can trigger a transfer move (see Transfer Move (Dialog)).
      • Show vehicle details: Open the vehicle dossier and switch to the Electrical process data tab to see more details.

      Notes

      • The SOC values displayed are based on telematics and may be delayed. Allow buffer time in planning.
      • SOC thresholds and alarm limits can be configured in the system settings or parameters (see Alarming & Thresholds).
      • For vehicles with hybrid powertrains, only the electrical components are considered.
