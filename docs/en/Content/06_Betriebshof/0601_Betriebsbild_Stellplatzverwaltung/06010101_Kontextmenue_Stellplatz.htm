      Using the “Parking Position” Context Menu

      The of a parking position allows you to perform quick maintenance and navigation actions in PEAK.DMS, e.g., blocking/unblocking, maintaining or allowed , and jumping to the graphical or tabular view.

      Opening the Context Menu

      1. Right-click a parking position in the .
      2. Select the desired action from the list.

      Actions at a Glance

      The available entries are project-specific. The following actions, among others, are available in the standard configuration:

      Menu EntryBrief Description
      Edit Parking Position…Opens the “Edit Parking Position” .
      BlockSets Blocked; optional reason for block/validity.
      UnblockLifts an existing block.
      Add Usage Type…Adds a to the parking position.
      Remove Usage TypeRemoves a selected .
      Allow Vehicle Type…Adds permitted /patterns.
      Remove Vehicle TypeRemoves type/pattern from the permission.
      Show in Depot ViewCenters and highlights the parking position.
      Show in “Parking Positions”Opens the list filtered for the parking position.
      Create Repositioning Move…Creates a (if enabled).
      HistoryShows the change log (if available).

      Typical Workflows

      1. Block Parking Position: Select Block → set reason for block and period → OK.
      2. Maintain Usage Type: Select Add Usage Type… → select type → optional period → OK.
      3. Update Permitted Vehicle Types: Select Allow Vehicle Type… → select type/pattern → OK.
      4. Quick Switch to List: Select Show in “Parking Positions” → check/edit record.
