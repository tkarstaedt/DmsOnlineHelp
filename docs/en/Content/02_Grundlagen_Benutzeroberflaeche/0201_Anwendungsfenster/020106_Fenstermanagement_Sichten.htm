      Window management — working with views

      In PEAK.DMS you can work flexibly with multiple views. You can split, move, maximize, duplicate, or close views. This lets you arrange your workspace to match your workflow.

      Split views

      You can split the display area so that several views are shown side by side or one above the other.

      • Drag a view’s tab with the mouse into the desired area (left, right, top, or bottom half of the screen).
      • Release the mouse button to place the view in a new subwindow.
      • Alternatively, choose View > Split window in the menu bar and select the split direction.

      Move views

      • Click the tab of the view.
      • Hold the mouse button and drag the view to the desired position.
      • Release the mouse button to place the view in the new subwindow.

      Maximize views

      • Double‑click the tab of the view, or
      • choose Maximize in the context menu.

      Double‑click again to return to the original layout.

      Duplicate views

      • Open the desired view again via the navigation tree or the toolbar.
      • Each instance appears as its own tab in the display area.

      Close views

      • Click the X on the tab, or
      • choose Close in the context menu.

      Context menus and tooltips

      Many functions for working with views are also available via the context menu (right‑click on tabs or display area). Hover over a button or icon to show helpful tooltips.

      Note: This chapter describes handling views as windows in the application window. Detailed information about the contents of views and how to use them can be found under Overview of views.
