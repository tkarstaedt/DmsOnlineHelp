      Use user profiles efficiently

      With user profiles you tailor your workspace in PEAK.DMS precisely and switch quickly between different tasks. The recommendations below help you create profiles systematically, switch quickly, and keep things tidy over time.

      Use profile variants

      • Role profiles: Create profiles for typical roles (e.g., Control Center, Workshop, Planning) to predefine suitable views and layouts.
      • Task profiles: Create profiles for recurring tasks (e.g., monitoring, analysis) to speed up context switching.
      • Naming conventions: Use descriptive names and optional version suffixes (e.g., “compact”, “v2”).

      Switch profiles quickly

      • Select at sign‑in: Choose the target profile in Your profile to start in the right environment immediately.
      • Switch during operation: Change profiles via the File menu when your focus changes (e.g., from dispatch to analysis).
      • Save current layout: Save your layout before switching so your adjustments are not lost.

      Maintain profiles

      • Update regularly: Adapt profiles to new workflows, views, or toolbars.
      • Remove legacy profiles: Delete outdated or rarely used profiles to keep the selection list lean.
      • Consolidate variants: Merge very similar profiles to avoid duplicate maintenance.

      Prepare the layout optimally

      • Arrange views: Use subwindows and tabs to place frequently combined views side by side.
      • Adjust columns: Show relevant columns, reorder them, and save the layout in the profile.
      • Curate toolbars: Surface the most important actions to reduce clicks.

      Avoid common issues

      • Unsaved changes: Save the profile after layout changes before switching profiles.
      • Delete conflicts: A loaded profile cannot be deleted. Load another profile first.
      • Unclear names: Include purpose and audience in profile names to prevent confusion.

      Checklist: Do’s & Don’ts

      Do’s Don’ts
      Structure profiles by role and task. Don’t assign arbitrary names without context.
      Save layout changes immediately into the profile. Don’t switch profiles with an unsaved layout.
      Clean up and consolidate regularly. Don’t keep outdated profiles for years.
      Tailor toolbars and views to the task. Don’t use overloaded layouts without prioritization.
