      Object interaction in views

      In PEAK.DMS views you directly access operational objects such as vehicles, drivers, or blocks. With various actions you can open details, run actions, or change data.

      Note: Fundamentals of filtering, sorting, and editing columns are in Views — data presentation. Information on customizing layouts and saving your settings is in Customize views.

      Select objects

      • Click a row or a graphical icon to select an object.
      • Multi‑selection is possible with the Ctrl or Shift key pressed.
      • The selection is highlighted.

      Use context menus

      Right‑click a selected object to open the context menu. Depending on the object type, different actions are available, for example:

      • Open details (e.g., driver dossier, vehicle dossier)
      • Start actions (e.g., record incident, create workshop order)
      • Export data

      Double‑click actions

      Double‑click a row or icon to open the corresponding detail dialog directly. Example: double‑click a driver to open the driver dossier.

      Actions from toolbars

      Many views also provide buttons in the view toolbar for starting frequent actions directly, such as creating new objects or editing selected objects.

      Examples of object interactions

      • Open vehicle dossier: Select a vehicle and use the context menu or double‑click to open the vehicle dossier dialog.
      • Delete driver: Select a driver and choose Delete in the context menu (only with appropriate permission).
      • Create workshop order: From a vehicle’s context menu, open the workshop order dialog.
