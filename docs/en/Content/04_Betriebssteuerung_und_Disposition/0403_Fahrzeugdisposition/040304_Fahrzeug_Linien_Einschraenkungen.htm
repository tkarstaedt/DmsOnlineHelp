      Dialog "Vehicle-Line Restrictions"

      Using the "Vehicle-Line Restrictions" dialog, you can define one or more lines for a vehicle on which it must be deployed. These restrictions affect the automatic and manual dispatching of blocks with a line reference.

      Overview

      The "Vehicle-Line Restrictions" dialog allows for the manual definition of line restrictions for individual vehicles. These define which lines a vehicle may be dispatched on.

      Each restriction is time-limited and can be given an optional reason. The system also displays all existing workshop and supply orders for the selected vehicle.

      Calling the Dialog

      1. Open the menu Edit → Vehicle-Line Restrictions.

      Dialog "Vehicle-Line Restrictions"

      Dialog Fields

      The dialog contains the following input fields:

      FieldDescription
      Vehicle Vehicle number of the vehicle for which line restrictions are being entered.
      The system displays all workshop and supply orders for this vehicle.
      Line Selection list with all available lines.
      Select the line on which the vehicle is permitted to be deployed.
      Valid until (Bis) Time until which the line restriction is valid.
      Default value: End of the next operating day (can be manually overwritten).
      Reason Optional free-text field for entering the reason or occasion for the restriction.

      Dialog "Vehicle-Line Restrictions" — Dialog Fields

      Operating Steps

      Entering a Line Restriction

      Entering a line restriction — step by step:

      1. Open the Vehicle-Line Restrictions dialog via the Edit menu.
      2. Vehicle field: Enter or select the vehicle number.
        → The system displays all workshop and supply orders for the vehicle.
      3. Line field: Select the desired line from the selection list.
      4. Valid until field: Check the suggested end date and adjust if necessary.
      5. Reason field (optional): Enter the reason for the restriction.
      6. Click the Apply button.
        → The line restriction is saved.
        → The system automatically provides another empty row.
      7. Enter further line restrictions if required (repeat steps 3-6).
      8. Close the dialog.

      Entering Multiple Restrictions

      You can enter multiple line restrictions for a vehicle in parallel. After each confirmation with Apply, a new input row is automatically made available.

      After successful entry, you can enter line restrictions for another vehicle.

      Effect in Dispatching

      The line restrictions affect vehicle and block dispatching as follows:

      Basic Rules

      • The line restrictions apply to all block segments of a block.
      • If a block contains several different lines and the vehicle is restricted to only one of these lines, the vehicle will not be dispatched to this block.

      Important: The restriction applies across the entire block. If a vehicle has been restricted to Line XX and a block contains block segments for Line XX and YY, then the vehicle will not be deployed for this block.

      Consideration of Workshop and Supply Orders

      • Workshop and supply orders are always respected without restriction.
      • A vehicle with a line restriction will only be dispatched to a corresponding block if no workshop or supply order begins before the block ends.

      Interaction with other Dispatching Rules

      Line restrictions interact with other rules from dispatching and duty lineup planning:

      • Permitted vehicle types
      • User-scheduled vehicles
      • Rules from duty lineup planning

      Example

      Scenario:

      A vehicle is exclusively approved for deployment on Line 20.

      Result:

      • Blocks that contain only Line 20 → Vehicle is dispatched
      • Blocks that contain both Line 20 and Line 21 → Vehicle is excluded

      Notes

      Use the "Reason" field to increase the traceability of line restrictions for other dispatchers (e.g., "Driver training Line 20", "Test drives").

      • Multiple line restrictions can be active for a vehicle simultaneously.
      • Each restriction has its own validity period.
      • Expired restrictions are automatically no longer considered.
