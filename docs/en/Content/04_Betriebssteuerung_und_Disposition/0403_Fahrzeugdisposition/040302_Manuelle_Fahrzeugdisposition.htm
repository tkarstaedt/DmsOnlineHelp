      Manual Dispatching

      Perform manual dispatching measures – assign vehicle to a block, dispatch stabling positions, swap vehicles, lock vehicles, and document exceptions. Results are visible in the Stabling Lineup Plan and Depot View.

      Overview

      Manual Dispatching includes targeted interventions in block and vehicle operations, including assigning vehicles to blocks/duties, Stabling Position Dispatching after pull-in, Vehicle Swap, Locking individual vehicles, and documenting exceptions.

      Prerequisites

      • Sign-in to the DMS client with appropriate Dispatching Permissions.
      • Timetable Data are up-to-date (day type, operating day, released version).

      Call and Context

      • Typical entry points: Block List, Vehicle List, Stabling Lineup Plan, Depot View.
      • Result display: Changes become visible in the Stabling Lineup Plan and the Depot View.

      Controls

      Assign vehicle to a block/duty

      1. Open the desired dispatching view (e.g., Block List or Vehicle List).
      2. Select the target Block/Duty or the target Vehicle.
      3. Start the action Assign Vehicle.
      4. Select a suitable vehicle (consider depot, class, status) and confirm the assignment.

      Perform vehicle swap

      1. Mark both affected entries (source/target).
      2. Select Swap Vehicle.
      3. Check departure times, range/energy, and locks; confirm and document the reason for the swap.

      Lock vehicle

      1. Open the Vehicle Details or Vehicle Dossier.
      2. Select Lock, enter reason and time frame, and confirm.

      Locked vehicles are marked accordingly in the views; dispatching considers the lock status.

      Start manual block dispatching (Overview)

      1. Open the Block Dispatching dialog and select Depot, Operating Day, and Pull-out Phase.
      2. Click on Start Dispatching and check the result in the Stabling Lineup Plan and Depot View.
      3. With Reset Blocks (Umläufe zurücksetzen) you undo assignments (blocks in status In Progress remain assigned).

      Stabling Position Dispatching after Pull-in

      The Stabling Position Dispatching enables the targeted assignment of vehicles to stabling positions, especially after pull-in or restoration of operational readiness. The dialog is part of Vehicle Dispatching.

      Call Vehicle Dispatching Dialog

      • Toolbar → Vehicle Dispatching
      • Context menu of a vehicle → Stabling Position Dispatching
      • Double-click in Block List or Pull-in List on a currently deployed vehicle

      Dialog Areas Vehicle Dispatching

      AreaDescription

      Check vehicle for operational readiness

      1. Open the Vehicle Dispatching dialog and select the vehicle in the Vehicle field.
      2. Check the current status in the Faults, Supply Orders, and Vehicle Reservations tables.
      3. Create new entries via New… if necessary.

      Report vehicle as operationally ready

      1. Click on Report Operational Readiness (Einsatzfähig melden) in the Dispatching area.
      2. The system automatically sets all faults and orders with status In Progress to Completed.
      3. Optional: Use Report Operational Readiness + Automatic Dispatching for a combined action (operational readiness report plus automatic stabling position assignment by the BMS).

      Note: A special User Permission (Benutzerrecht) is required for this action.

      Automatic vehicle dispatching

      1. Click on Automatic Dispatching.
      2. The BMS calculates a suitable Stabling Position (Abstellplatz) based on the stabling lineup plan.
      3. The suggestion is displayed in the Depot and Stabling Position (Stellplatz) fields.
      4. You can reject the suggestion with Delete Target Location or accept it by confirming.
      5. After a parameterized time (e.g., 90 seconds), the suggestion is automatically accepted.

      Note: If no stabling position is found, an information dialog appears regarding possible User-scheduled Vehicle Pre-assignments (Wunschvorbelegungen) (block reference).

      Manual vehicle dispatching

      1. Click on Manual Dispatching.
      2. The Set Target Location dialog opens.
      3. Select Depot, Track/Stabling Position, and optionally Section (Abschnitt).
      4. Confirm with OK. The stabling position is set as the planned location.

      Special features for traction vehicles

      • Traction Vehicles are evaluated individually.
      • The Traction Status corresponds to the most severe individual status of all coupled vehicles.
      • Only dispatch tractions as a whole if all coupled vehicles are actually in the same depot.

      Documentation of Exceptions

      • Record a Comment (Kommentar) with date/role/reason.
      • Link relevant Messages and Work Orders (Werkstatt/Versorgung).
      • Upon reversal: restore the original status and justify the reason.

      Notes & Best Practices

      • Before swap and assignment, check Range, Energy/Charging Window (Ladefenster), and Deadlines (Fristen).
      • Comply with standard rules (e.g., permissible vehicle types, line restrictions); manual intervention only when necessary.
      • Coordinate conflicts with Workshop/Supply, consider locks and orders.
      • Always synchronize dispatching with the real Vehicle Status.
      • Faults and Open Orders must be resolved or documented before reporting operational readiness.
      • When using Automatic Dispatching, workshop/supply locks are considered.
      • Do not manually set vehicles with ongoing dispatching or those currently in motion.
      • Repeated dispatching with the same parameters yields the same result, provided no changes have occurred in the system.
