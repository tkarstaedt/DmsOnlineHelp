Datensätze filtern

Mit Filtern grenzen Sie in tabellarischen Sichten von PEAK.DMS die angezeigten Datensätze auf eine Teilmenge ein. So blenden Sie Unwichtiges aus und konzentrieren sich auf die relevanten Informationen.

Verweis: Grundlagen zu Sortieren, Suchen und Spaltenanpassung finden Sie in Arbeiten mit Daten in Sichten und Datensätze suchen .

Filterarten

  • Einfache Filter: arbeiten spaltenweise über eine Filterzeile, vergleichbar mit Filtern in MS Excel.
  • Komplexe Filter: kombinieren mehrere Bedingungen und Verknüpfungen in einem eigenen Bereich oberhalb der Tabelle.

Einfache Filter verwenden

  1. Klicken Sie in der Werkzeugleiste auf das Symbol für Filterzeile einblenden . Unterhalb der Spaltenköpfe erscheint eine leere Zeile.
  2. Geben Sie in der Filterzeile in der gewünschten Spalte das Filterkriterium ein. Die Tabelle wird sofort gefiltert.

Abbildung: Einfache Filter aktivieren und Kriterium eingeben

Operatoren je Spalte wählen

Die verfügbaren Vergleichsoperatoren hängen vom Datentyp der Spalte ab (Text, Zahl, Datum). Öffnen Sie die Operatorauswahl über das Symbol am linken Rand der Filterzelle.

  • Textspalten: enthält, beginnt mit, endet mit, gleich, ungleich
  • Numerische Spalten: =, ≠, <, ≤, >, ≥, Bereich
  • Datums-/Zeitspalten: am, vor, nach, zwischen

Filter im Spaltenkopf verwalten

Ist eine Spalte gefiltert, erscheint ein Filtersymbol im Spaltenkopf. Klicken Sie darauf, um Details zu bearbeiten.

  • Operator im Drop-down ändern
  • Kriterium im Eingabefeld anpassen
  • Filter entfernen , um die Spalte zu bereinigen
  • Dialog mit Schließen beenden

Abbildung: Zusätzlicher Filterdialog für eine Spalte

Hinweis: Einfache Filter sind spaltenweise definiert und wirken UND-verknüpft . Mehrere Spaltenkriterien können somit kombiniert werden.

Komplexe Filter verwenden

Mit komplexen Filtern verknüpfen Sie mehrere Bedingungen logisch und bilden auch hierarchische Strukturen ab.

  1. Klicken Sie in der Symbolleiste auf Erweiterter Filter . Der Filterbereich wird oberhalb der Tabelle eingeblendet.
  2. Bestehende einfache oder komplexe Filter werden dort grafisch dargestellt und können erweitert oder geändert werden.

Abbildung: Komplexer Filterbereich

Visuellen Filter-Editor verwenden

  1. Klicken Sie auf Filter bearbeiten , um den visuellen Editor zu öffnen.
  2. Definieren Sie den Filter per Mausklick auf Gruppen, Bedingungen, Spalten, Operatoren und Werte.
  3. Übernehmen Sie den Filter mit OK . Die Darstellung erscheint im Filterbereich und wird auf die Tabelle angewendet.

Abbildung: Visueller Filter-Editor

Aufbau komplexer Filter

  • Gruppen: Verknüpfen Bedingungen mit UND oder ODER . Gruppen bilden die Knoten der Baumstruktur.
  • Bedingungen: Einträge unterhalb einer Gruppe. Jede Bedingung besitzt Spalte , Operator , Wert .
  • Klicken Sie auf das + hinter einer Gruppe, um eine Bedingung hinzuzufügen.
  • Klicken Sie auf das x an Gruppe oder Bedingung, um diese zu entfernen.
  • Klicken Sie auf Spalte , Operator oder Wert , um die Auswahl bzw. Eingabe zu ändern.

Filter temporär deaktivieren oder verwerfen

  • Deaktivieren Sie den Filter vorübergehend über die Checkbox im Filterbereich.
  • Verwerfen Sie den Filter dauerhaft über das X neben der Filterdefinition.

Abbildung: Zusatzfunktionen im Filterbereich

Filter als Text bearbeiten

Im Dialog Filter bearbeiten können Sie optional den Filter-String direkt als Text editieren. Änderungen am Text und an den visuellen Elementen werden gegenseitig synchronisiert. Syntaxkonforme Anpassungen werden sofort in der Tabelle wirksam.

Abbildung: Filter-String bearbeiten

Tipps: Prüfen Sie bei ausbleibenden Treffern, ob zusätzlich eine Suche aktiv ist. Übernehmen Sie bewährte Filter als Startpunkt, indem Sie sie im erweiterten Filterbereich wiederverwenden.
